Pathfinder 패스파인더

여러 개체를 선택한 뒤 한 가지 계산 방식으로 결과를 만드는 기능입니다. 실행 후 결과가 자동으로 그룹처럼 묶이는 흐름을 기억합니다.

Unite 합치기

겹친 여러 도형을 하나의 닫힌 면으로 합칩니다.

Minus Front 빼기

앞면 개체를 기준으로 뒤쪽 도형에서 겹친 부분을 제외합니다.

Divide 나누기

겹치는 경계를 기준으로 조각을 나눕니다. 필요 없는 조각은 선택해서 정리합니다.

  • 선택을 정확하게 해야 원하는 결과가 나옵니다.
  • 실행 후 개체가 자동으로 그룹처럼 묶이는 흐름을 기억합니다.

Shape Builder 도형 구성 도구

선택된 도형 안에서 필요한 부분을 직접 더하고, 빼고, 나눠서 빠르게 모양을 정리합니다.

여러 기능을 한 번에 판단
  • + 더하기드래그해서 이어 붙입니다.
  • − 빼기Alt 클릭 또는 Alt 드래그로 제거합니다.
  • ÷ 나누기필요한 영역을 클릭해서 독립된 조각으로 만듭니다.
  • ! 선택 먼저편집할 모양을 정확하게 선택해야 도구가 작동합니다.
