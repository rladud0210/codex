수업 핵심

오늘 꼭 남겨야 하는 4가지

  • Ctrl + Y로 윤곽선 모드를 확인하면 숨은 개체와 구조를 빠르게 읽을 수 있습니다.
  • Ctrl + A와 Ctrl + Alt + A의 차이를 구분해야 전체 선택 실수가 줄어듭니다.
  • 레이어 안에서 앞뒤 순서 바꾸기는 단축키로 가능하지만, 다른 레이어로 이동은 패널 드래그만 가능합니다.
  • 복제는 한 가지 방식만 쓰지 말고, 상황에 따라 붙여넣기 위치를 구분해서 써야 합니다.
빠른 암기 문장
  • 편집 전에는 “기본형태가 뭐였지?”를 먼저 묻기
  • 순서는 “뒤에서 앞으로”
  • 작업 흐름은 “메인에서 디테일”
  • 선택이 꼬이면 “윤곽선 모드부터 확인”

편집 전 체크

작업이 망가지는 이유는 대부분 편집을 빨리 시작해서입니다. 수정 전에 아래 3가지를 반드시 점검하세요.

작업 습관
편집 전 체크!
  1. 기본 형태가 뭐였을까?
    처음 실루엣과 구조를 먼저 떠올리고 수정해야 원형이 무너지지 않습니다.
  2. 뒤에서 앞으로
    배경이나 뒷면부터 순서를 잡고, 앞쪽 요소를 나중에 정리하면 겹침이 덜 꼬입니다.
  3. 메인에서 디테일
    큰 비율과 형태를 먼저 잡고, 마지막에 작은 장식과 세부를 손봅니다.
수업 주의

이럴 때 작업이 자주 꼬입니다

  • 선택이 제대로 안 됐는데 바로 드래그하는 경우
  • 숨은 개체를 확인하지 않고 지우거나 이동하는 경우
  • 앞뒤 순서를 보지 않고 복제만 반복하는 경우
  • 다른 레이어 이동도 단축키로 될 거라고 생각하는 경우

선택 기능 정리

선택 도구는 단순히 고르는 기능이 아니라, 작업의 시작점입니다. 어떤 방식으로 선택하느냐에 따라 편집 속도와 정확도가 달라집니다.

선택 정확도
도구

기본 선택 도구

자주 사용
  • Selection Tool: 오브젝트 전체를 선택하고 이동할 때 사용
  • Direct Selection Tool: 점, 선, 일부 패스를 직접 선택해 수정할 때 사용
  • Group Selection Tool: 그룹 안에서 필요한 개체를 단계적으로 선택할 때 도움
  • Magic Wand: 같은 속성의 개체를 한 번에 선택할 때 사용
기억 포인트
전체를 건드릴 때는 Selection, 일부 점이나 선을 고칠 때는 Direct Selection입니다.
선택 보조

놓치기 쉬운 선택 기능

시험 대비
  • Enter: 수치 입력이나 이동값 설정 같은 작업과 연결해서 자주 씀
  • Shift: 선택 추가 또는 선택 해제 보조에 자주 사용
  • Ctrl + 클릭: 현재 모드에서 빠르게 선택 상태를 바꿀 때 활용
  • Select > Same: 같은 속성의 개체를 한 번에 잡아낼 때 유용
숨은 개체 찾기

보이지 않는 개체 선택하기

실수 방지
  • Shift 선택 해제: 여러 개 중 일부만 빼고 싶을 때 사용
  • Ctrl + Y: 윤곽선 모드로 전환해 가려진 구조와 숨은 개체를 확인
  • Ctrl + 클릭: 겹친 상태에서 원하는 개체를 다시 겨냥
핵심
선택이 이상하면 먼저 Ctrl + Y로 윤곽선 모드를 켜서 구조부터 확인하세요.
전체 선택

범위를 구분해야 하는 단축키

구분 필수
  • Ctrl + A: 전체 선택
  • Ctrl + Alt + A: 아트보드 내 개체 전체 선택
  • Ctrl + 6: 다시 선택
자주 나오는 차이
모든 개체를 잡을지, 현재 아트보드 안의 개체만 잡을지 먼저 생각하고 눌러야 합니다.

레이어와 순서 변경

앞뒤 순서는 단축키로 빠르게 정리할 수 있지만, 레이어 간 이동은 방식이 다릅니다. 이 차이를 정확히 구분하세요.

꼭 구분

다른 레이어로 이동은 단축키로 불가능

현재 메모 기준에서 중요한 점은 이것입니다. 다른 레이어로 이동하고 싶다면 레이어 패널을 열고 직접 드래그해야 합니다. 앞뒤 순서 변경과 레이어 이동을 같은 개념으로 생각하면 실수가 생깁니다.

  • 같은 레이어 안 순서 변경: 단축키 가능
  • 다른 레이어로 옮기기: 패널에서 직접 드래그
