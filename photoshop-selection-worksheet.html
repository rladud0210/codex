수업 포인트

도구 이름을 외우는 것보다 어떤 기준으로 선택하는지 이해해야 실제 사진에서 빠르게 고를 수 있습니다.

핵심 흐름

큰 선택 → 부족한 부분 추가 → 넘친 부분 빼기 → 가장자리 정리 → 새 레이어 복제

학습 방식

설명 확인 → 문제 풀이 → 채점 → 정답 보기 → 실습 체크 순서로 복습합니다.

빠른 개요

  • Object Selection은 개체를 자동 인식합니다.
  • Quick Selection은 색과 질감을 문지르듯 선택합니다.
  • Magic Wand는 클릭한 색상과 비슷한 영역을 선택합니다.
  • Lasso는 직접 그려 선택 영역을 만듭니다.
Selection Tools

선택 도구 핵심 정리

선택 도구는 대상의 형태, 색, 경계가 얼마나 뚜렷한지에 따라 골라 써야 합니다.

Object Selection

개체 기준

  • 사람, 상품, 물건처럼 형태가 분명한 개체를 빠르게 선택합니다.
  • 처음 큰 영역을 잡을 때 좋습니다.
  • 선택 후 Quick Selection으로 세부 보정하면 효율적입니다.
Quick Selection

색과 질감 기준

  • 문지르듯 드래그해서 비슷한 색과 질감의 영역을 선택합니다.
  • [, ]로 브러쉬 크기를 조절합니다.
  • Shift는 추가, Alt는 빼기입니다.
Magic Wand

클릭한 색 기준

  • 클릭한 색상과 비슷한 영역을 한 번에 선택합니다.
  • 단색 배경이나 색 차이가 분명한 이미지에 적합합니다.
  • 복잡한 사진에서는 선택이 거칠 수 있습니다.

Lasso Tool

직접 그려서 자유롭게 선택 영역을 만듭니다. 자동 선택이 잘 안 되는 작은 영역을 손으로 보정할 때 좋습니다.

Caps Lock

브러쉬 커서 표시가 바뀝니다. 커서가 갑자기 십자 모양처럼 보이면 Caps Lock 상태를 확인합니다.

Shortcuts

자주 쓰는 단축키

선택 영역

  • 선택 해제: Ctrl + D
  • 선택 영역 반전: Ctrl + Shift + I
  • 선택 영역 새 레이어 복제: Ctrl + J

변형과 브러쉬

  • 자유 변형: Ctrl + T
  • 변형 적용: Enter
  • 브러쉬 축소: [
  • 브러쉬 확대: ]

Shift

  • 선택 도구에서는 선택 영역 추가와 다중 선택입니다.
  • 자유 변형에서는 정비율 유지입니다.

Alt

  • 선택 도구에서는 선택 영역 빼기입니다.
  • 자유 변형에서는 가운데 기준 크기 조절입니다.
Modify Selection

선택 영역 수정

선택은 처음부터 완벽하게 잡는 것보다 잡은 뒤 수정하는 과정이 중요합니다.

기능의미사용 상황
Ctrl + 레이어 썸네일 클릭그 레이어의 불투명 영역을 그대로 선택 영역으로 불러옵니다.이미 분리된 레이어 모양을 다시 선택할 때
Select > Modify > Expand선택 영역 확대선택이 안쪽으로 부족하게 잡혔을 때
Select > Modify > Contract선택 영역 축소선택이 바깥쪽으로 넘쳤을 때
Select > Modify > Feather가장자리 흐리게합성 경계를 부드럽게 만들 때
Select and Mask털, 머리카락, 복잡한 가장자리 정리일반 선택 도구로 경계가 지저분할 때
Color Range

Color Range 이해

Color Range는 색을 기준으로 선택 영역을 만드는 기능입니다. 미리보기의 흰색, 회색, 검은색 의미를 반드시 구분해야 합니다.

흰색

선택 영역으로 잡힙니다.

회색

반투명하게 선택됩니다.

검은색

선택되지 않습니다.

Worksheet

복습 문제

빈칸에 직접 입력하고, 아래 버튼으로 채점과 정답 확인까지 할 수 있습니다.

빈칸 채우기

1. 개체를 자동으로 인식해서 선택하는 도구는 이다.
2. 색과 질감을 문지르듯 드래그해서 선택하는 도구는 이다.
3. 클릭한 색상과 비슷한 영역을 선택하는 도구는 이다.
4. 직접 그려서 자유롭게 선택 영역을 만드는 도구는 이다.
5. 선택 영역 해제 단축키는 이다.
6. 선택한 영역을 새 레이어로 복제하는 단축키는 이다.
7. 선택 영역 반전 단축키는 이다.
8. 자유 변형은 , 적용 완료는 이다.
점수: 아직 채점 전

개념 확인

1. Shift 키가 선택 도구와 자유 변형에서 각각 어떤 역할을 하는지 쓰세요.
예시 답: 선택 도구에서는 선택 영역 추가와 다중 선택, 자유 변형에서는 정비율 유지 역할을 한다.
2. Alt 키가 선택 도구와 자유 변형에서 각각 어떤 역할을 하는지 쓰세요.
예시 답: 선택 도구에서는 선택 영역 빼기, 자유 변형에서는 가운데 기준 크기 조절 역할을 한다.
3. Quick Selection Tool과 Magic Wand Tool의 차이를 설명하세요.
예시 답: Quick Selection은 드래그하며 색과 질감의 경계를 따라 선택하고, Magic Wand는 클릭한 색상과 비슷한 영역을 한 번에 선택한다.
4. Select and Mask는 언제 사용하나요?
예시 답: 머리카락, 털, 복잡한 가장자리처럼 일반 선택 도구만으로 깔끔하게 잡기 어려운 부분을 정리할 때 사용한다.

상황 판단 문제

1. 흰 배경 위 상품만 빠르게 선택하려면 어떤 도구를 먼저 쓰는 것이 좋은가?
예시 답: Object Selection Tool로 상품을 먼저 선택하고, 부족한 부분은 Quick Selection Tool로 보정한다.
2. 파란 하늘만 선택해 색을 바꾸려면 어떤 방법이 적합한가?
예시 답: Magic Wand Tool 또는 Color Range처럼 색을 기준으로 선택하는 방법이 적합하다.
3. 인물을 선택했지만 배경을 수정해야 한다면 어떤 단축키를 쓰는가?
예시 답: Ctrl + Shift + I로 선택 영역을 반전한다.

실습 체크 문제

1. Object Selection Tool로 큰 개체를 선택해보세요.
2. Quick Selection Tool로 부족한 부분을 추가해보세요.
3. Alt를 누른 채 넘친 선택 영역을 빼보세요.
4. Select and Mask로 가장자리를 정리해보세요.
5. Ctrl + J로 선택 영역을 새 레이어로 복제해보세요.
6. Ctrl + T로 크기와 위치를 조정한 뒤 Enter로 적용해보세요.

정답 요약

  • Quick Selection: 문지르듯 드래그
  • Magic Wand: 클릭한 색상 기준
  • Lasso: 직접 그려 선택

단축키

  • 선택 해제: Ctrl + D
  • 선택 복제: Ctrl + J
  • 자유 변형: Ctrl + T
  • 선택 반전: Ctrl + Shift + I

선택 수정

  • Expand: 확대
  • Contract: 축소
  • Feather: 가장자리 흐리게
  • Select and Mask: 털과 머리카락 정리

Color Range

  • 흰색: 선택됨
  • 회색: 반투명 선택
  • 검은색: 선택 안 됨
Practice Flow

실습 흐름

Object Selection Tool로 개체를 먼저 선택하고, Quick Selection Tool로 부족한 부분을 추가합니다. 넘친 부분은 Alt로 빼고, Select and Mask에서 머리카락과 털을 정리합니다. 마지막으로 Ctrl + J로 새 레이어를 만들고 Ctrl + T로 크기와 위치를 조정합니다.
