정답지

객관식 / 빈칸

  1. Object Selection Tool
  2. Magic Wand Tool
  3. Quick Selection Tool
  4. Lasso Tool
  5. Ctrl + D
  6. Ctrl + J
  7. Ctrl + T
  8. Ctrl + Shift + I
  9. [, ]
  10. Caps Lock

Shift / Alt

  • Shift 선택 도구: 선택 영역 추가, 다중 선택
  • Shift 자유 변형: 정비율 유지
  • Alt 선택 도구: 선택 영역 빼기
  • Alt 자유 변형: 가운데 기준 크기 조절

연결 문제

  1. Expand - C
  2. Contract - E
  3. Feather - A
  4. Select and Mask - D
  5. Ctrl + 레이어 썸네일 클릭 - B

Color Range

  • 흰색: 선택 영역으로 잡힘
  • 회색: 반투명하게 선택됨
  • 검은색: 선택되지 않음

상황별 도구

  1. Object Selection Tool
  2. Magic Wand Tool 또는 Color Range
  3. Select > Select and Mask 또는 옵션바의 Select and Mask

실습 순서

  1. Object Selection Tool로 큰 개체 선택
  2. Quick Selection Tool과 Alt로 보정
  3. Select and Mask로 가장자리 정리
  4. Ctrl + J로 새 레이어 복제
  5. Ctrl + T로 변형 후 Enter 적용

서술형 예시

  • Quick Selection은 문지르듯 드래그해 색과 질감의 경계를 따라 선택하고, Magic Wand는 클릭한 색상과 비슷한 색을 한 번에 선택한다.
  • 배경을 선택한 뒤 인물만 남기고 싶거나, 인물을 선택한 뒤 배경을 보정하고 싶을 때 선택 영역을 반전한다.
