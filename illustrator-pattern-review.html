1. 패턴 생성 핵심 정리

패턴은 복잡한 그림이 아니라, 반복 기준이 되는 최소 견본을 정확하게 만드는 것부터 시작합니다.

수업 핵심 순서
기본 흐름

최소 견본으로 패턴 만들기

  • 1
    패턴으로 반복될 최소 견본을 만든다. 기본 시작 예시는 사각형이다.
  • 2
    완성한 견본을 Swatches 패널에 저장한다.
  • 3
    패턴 썸네일을 더블클릭하면 수정 모드로 들어갈 수 있다.
  • 4
    필요하면 Size Tile to Art를 사용해 타일 크기를 아트에 맞춘다.
기억 포인트

Swatches 패널에서 확인할 것

  • 핵심
    패턴은 작업 화면의 개체만 보는 것이 아니라 Swatches에 저장된 상태까지 확인해야 한다.
  • 핵심
    썸네일 더블클릭은 단순 미리보기가 아니라 패턴 수정 진입 기능이다.
  • 핵심
    Size Tile to Art를 이해하면 타일 영역과 실제 반복 범위를 더 쉽게 맞출 수 있다.
한 줄 정리: 패턴은 “최소 견본 제작 → Swatches 저장 → 썸네일 더블클릭으로 수정” 흐름으로 기억하면 정리가 쉽습니다.

2. 패턴 수정과 변형 이해

Ctrl + K 환경설정과 변형 옵션을 이해하면 개체와 패턴이 어떻게 움직이는지 구분할 수 있습니다.

개체 vs 패턴 구분

켜짐 개체와 패턴이 함께 크기 조절된다.
꺼짐 개체만 조절되고, 내부 패턴은 그대로 유지된다.
변형 도구

Transform 도구에서 구분할 옵션

  • 1
    Transform Pattern: 패턴 자체를 변형할 때 사용
  • 2
    Transform Object: 외곽 개체를 변형할 때 사용
  • 3
    두 옵션을 같이 보면서 “무엇이 움직여야 하는지” 먼저 판단한다.
암기 문장

Pattern Tile이 켜지면 같이, 꺼지면 개체만.

암기 문장

Transform Pattern은 무늬, Transform Object는 틀.

암기 문장

수정은 썸네일 더블클릭부터 시작.

3. 이미지 관리와 벡터화

패턴 수업과 함께 자주 이어지는 이미지 작업 흐름도 같이 정리해 두면 실습 속도가 빨라집니다.

실무형 보조 흐름
이미지 관리

기본 작업 순서

  • 1
    Ctrl + O로 이미지를 열고, 필요하면 새 문서를 Ctrl + N으로 만든다.
  • 2
    배치된 이미지는 Embed로 포함시켜 링크 문제를 줄인다.
  • 3
    이미지 위에서 우클릭 후 Crop Image로 필요한 영역만 정리한다.
이미지 벡터화

Image Trace

비트맵 이미지를 벡터 느낌으로 바꾸고 싶을 때는 Image Trace(이미지 추적)를 사용한다.

  • 팁
    원본 이미지를 먼저 정리한 뒤 추적하면 결과가 더 깔끔하다.
  • 팁
    크롭 후 추적하면 불필요한 배경까지 함께 벡터화되는 일을 줄일 수 있다.
빠른 복습: 열기 → 새 문서 → Embed → Crop Image → Image Trace 순서로 묶어 기억하면 실습에서 바로 연결됩니다.
