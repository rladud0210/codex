View Control

환경설정과 화면 조작

작업 시작 전에 화면을 빠르게 다루는 습관을 익히면 도형 수정, 패스 확인, 배치 작업 속도가 확실히 빨라집니다.

환경설정 열기

Ctrl K

Illustrator의 기본 설정 창을 여는 단축키입니다.
단위, 선택 옵션, 인터페이스, 파일 처리 설정을 바꿀 때 사용합니다.

수업 중 설정 안내가 나오면 가장 먼저 Ctrl + K로 환경설정을 엽니다.

확대 / 축소

Alt + 마우스 휠 회전

도형 디테일을 볼 때는 확대,
전체 배치를 볼 때는 축소합니다.
손 도구로 전환하지 않고도 빠르게 시야를 조절할 수 있습니다.

확대와 축소는 선택 상태와 무관하게 자주 반복되므로 손에 익혀두는 것이 좋습니다.

화면 이동 1

Space + 마우스 왼쪽 클릭 드래그

캔버스를 손으로 잡아 끌듯 이동합니다. 세밀한 편집 중 가장 많이 쓰는 방식입니다.

화면 이동 2

마우스 휠 꾹 누르기 이동

키보드 없이도 화면을 드래그할 수 있습니다. 마우스 중심 작업자에게 편한 방식입니다.

실습 팁

확대/축소와 화면 이동을 섞어 쓰면 아트보드 전체와 세부 편집 구간을 빠르게 오갈 수 있습니다.

  • 배치 확인: 축소
  • 정렬 수정: 화면 이동
  • 세부 조정: 확대
Save & Export

저장과 내보내기 차이 이해하기

작업 파일 보관과 제출용 결과물 저장은 목적이 다릅니다. 저장 메뉴와 Export 메뉴를 구분해서 사용해야 합니다.

File → Save / Save As / Save a Copy

  • Save: 현재 작업 파일을 같은 이름으로 저장합니다.
  • Save As: 다른 이름이나 다른 형식으로 새 작업 파일을 저장합니다.
  • Save a Copy: 원본 작업 파일은 유지한 채 복사본을 따로 저장합니다.
일러스트 파일과 PDF 파일을 작업용으로 보관할 때는 Save 계열 메뉴를 먼저 구분해서 사용합니다.

PDF를 다시 편집하려면

Illustrator에서 PDF를 저장한 뒤 다시 열어 편집하려면 아래 옵션이 중요합니다.

PDF 저장 옵션 → Preserve Illustrator Editing Capabilities
이 옵션을 켜 두면 Illustrator 내부 편집 정보가 함께 저장되어, 추후 PDF를 일러스트에서 수정하기 쉬워집니다.

File → Export → Export As / Save for Web

완성본을 이미지나 웹용 파일로 내보낼 때는
저장이 아니라 Export를 사용합니다.

  • Export As: 아트보드 기준으로 PNG, JPG 등 일반 결과물을 만들 때 사용합니다.
  • Save for Web: 웹 업로드용 최적화가 필요할 때 사용합니다.

Export As 체크 포인트

  • Use Artboards에 체크합니다.
  • 필요한 아트보드만 내보내려면 Range를 정확히 입력합니다.
  • 아트보드 기반으로 결과물이 잘려 저장되므로 제출용 파일 정리에 유리합니다.
Artboard

아트보드 기준으로 내보내기

원문에서 따로 적어주신 아트보드 항목을 독립해서 정리했습니다. 내보내기 할 때는 아트보드 기준으로 파일이 잘리는지 반드시 확인합니다.

체크할 항목

  • File → Export → Export As로 들어갑니다.
  • Use Artboards에 체크합니다.
  • Range에 내보낼 아트보드 번호를 입력합니다.
아트보드를 정확히 잡아두면 불필요한 여백 없이 결과물이 정리됩니다.

Range 입력 차이

두 표기는 비슷해 보여도 결과가 다릅니다.
수강생이 가장 자주 헷갈리는 부분이라
별도로 강조해 두는 것이 좋습니다.

1,3 → 1번, 3번만
1-3 → 1번부터 3번까지 전체
입력 방식 의미 예시 해석
1,3 쉼표로 구분된 특정 아트보드만 선택 1번과 3번 아트보드만 각각 출력
1-3 하이픈으로 연결된 연속 범위 전체 선택 1번, 2번, 3번 아트보드를 모두 출력
Selection

선택 기본

오브젝트를 정확히 고르고 필요한 대상만 다루는 것은 모든 실습의 기본입니다. 특히 Shift 동작을 익히는 것이 중요합니다.

정비율 유지

Shift

크기 조절 중 Shift를 함께 사용하면 비율을 유지한 채 확대/축소할 수 있습니다.

다중선택

Shift

이미 선택된 상태에서 다른 오브젝트를 추가로 클릭하면 여러 개를 동시에 선택할 수 있습니다.

선택 해제

Shift

다중선택 상태에서 특정 오브젝트를 다시 클릭하면 그 오브젝트만 선택에서 제외할 수 있습니다.

정비율 유지, 다중선택, 선택 해제까지 모두 Shift 하나로 자주 해결됩니다.
Duplicate

복사와 제자리 복제 단축키

오브젝트를 배치하거나 변형 버전을 만들 때 가장 많이 사용하는 복사 방식입니다. 앞뒤 레이어 위치까지 함께 기억해 두면 좋습니다.

기본 복사 / 붙여넣기

Ctrl C Ctrl V

가장 일반적인 복사 방식입니다. 새 위치에 복제본을 만들어 사용할 때 적합합니다.

드래그 복제

Alt + 드래그

오브젝트를 이동시키는 동시에 복제합니다. 반복 배치할 때 속도가 빠릅니다.

제자리 복제

CtrlC → CtrlF

같은 위치에 바로 앞쪽으로 복제합니다. 채우기 색만 바꾸거나 효과를 겹쳐 적용할 때 유용합니다.

앞으로 복제

CtrlC → CtrlF

복사한 오브젝트가 같은 위치의 앞에 쌓입니다.

뒤로 복제

CtrlC → CtrlB

복사한 오브젝트가 같은 위치의 뒤에 쌓입니다.
