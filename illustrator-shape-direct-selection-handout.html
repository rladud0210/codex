도형 그리기와 직접선택도구, 오늘 수업 핵심만 정리

오늘 배운 내용은 도형을 그리고, 필요한 포인트만 골라 형태를 조정하는 흐름입니다. 단축키를 익히고, 패스와 앵커 포인트의 개념을 이해하면 더 빠르고 정확하게 편집할 수 있습니다.

복습 목적: 보고 끝내는 자료가 아니라, 직접 써보며 기억하는 학습지
오늘의 핵심 1

도형을 그린 뒤 직접선택도구로 특정 포인트만 골라 다듬는 흐름이 중요합니다.

오늘의 핵심 2

Shift, Ctrl, 그룹과 잠금 단축키를 알면 작업 속도가 크게 빨라집니다.

오늘의 핵심 3

열린 패스와 닫힌 패스를 구분해야 어떤 모양이 만들어지는지 예측할 수 있습니다.

오늘의 핵심 4

격리모드는 그룹을 풀지 않고도 원하는 개체만 수정할 수 있게 도와주는 안전한 편집 모드입니다.

단축키 보기 워크시트 풀기 수업 체크리스트

자주 쓰는 단축키

오늘 수업에서 바로 자주 쓰게 되는 단축키들입니다. 외우려 하기보다 반복하면서 손에 익히는 것이 더 중요합니다.

Ctrl + Z
실행 취소

방금 한 작업을 뒤로 되돌립니다.

Ctrl + Shift + Z
재실행

되돌린 작업을 다시 앞으로 가져옵니다.

Shift
보조 키
  • 정비율 유지
  • 다중 선택
  • 선택 해제 보조
Ctrl + D
이전 작업 반복

방금 했던 변형이나 복제 작업을 한 번 더 반복합니다.

Ctrl + G
그룹

여러 개체를 한 덩어리처럼 묶어 함께 이동하거나 정리할 수 있습니다.

Ctrl + Shift + G
그룹 해제

묶여 있던 그룹을 다시 풀어 각각의 개체로 만듭니다.

Ctrl + 2
잠금

선택한 개체를 움직이지 않게 잠가 둡니다.

Ctrl + Alt + 2
전체 잠금 해제

잠겨 있던 모든 개체를 한 번에 다시 편집 가능하게 만듭니다.

패스와 직접선택도구 이해하기

도형 편집은 결국 패스를 이해하는 일입니다. 어떤 점을 선택하고, 어떻게 움직일지 생각하면서 편집해야 형태가 깔끔해집니다.

패스 Path

패스는 벡터로 이루어진 선입니다. 앵커 포인트와 선분으로 연결되어 있어, 확대해도 깨지지 않고 다시 수정할 수 있습니다.

벡터 선앵커 포인트재편집 가능

열린 패스 / 닫힌 패스

시작점과 종료점이 다르면 열린 패스, 같으면 닫힌 패스입니다. 면이 생기려면 보통 닫힌 패스가 필요합니다.

열린 패스: 시작점 ≠ 종료점닫힌 패스: 시작점 = 종료점

도구 활용과 실전 감각

오늘 배운 기능은 실제 작업에서 서로 연결해서 사용합니다. 상황과 목적을 같이 기억해 두면 다음 작업에서 훨씬 빠르게 꺼내 쓸 수 있습니다.

Direct Selection Tool

직접선택도구는 전체 개체가 아니라 원하는 포인트만 선택해서 수정하는 도구입니다.

  • 원하는 앵커 포인트만 선택
  • 패스 일부만 이동
  • 핸들을 조절해 곡률 수정
핵심: 도형 전체를 움직이는 것이 아니라, 모양을 이루는 점과 선을 고른 뒤 편집합니다.

편집 전에 먼저 생각할 것

  • 이 도형의 기본 원형은 무엇이었을까?
  • 어느 앵커 포인트를 선택해야 원하는 방향으로 변할까?
  • 직선으로 만들 것인지, 곡선으로 만들 것인지?
모양이 이상하게 무너지면, 너무 많은 포인트를 한꺼번에 움직였는지 먼저 확인합니다.

도형 그리기 팁

  • 도형 그리기 전에 기존 개체를 먼저 선택하면 같은 색상으로 바로 그릴 수 있습니다.
  • Shift를 누르면 원, 정사각형처럼 정비율을 유지하기 쉽습니다.
  • 도형을 만든 뒤 직접선택도구로 일부 포인트만 수정하면 새로운 형태를 빠르게 만들 수 있습니다.

격리모드

원하는 개체만 따로 편집할 수 있는 기능입니다.

  • 그룹을 풀지 않고 내부 개체만 수정하고 싶을 때 사용
  • 실수로 다른 개체를 건드릴 가능성을 줄일 수 있음
  • 해제할 때는 여백을 더블클릭

그룹과 잠금의 차이

  • 그룹은 여러 개체를 함께 움직이기 쉽게 묶는 기능
  • 잠금은 해당 개체를 실수로 선택하거나 이동하지 않게 막는 기능
  • 배경이나 기준선은 잠가두고 작업하면 훨씬 안정적입니다.

반복 작업 줄이기

  • Ctrl + D로 방금 한 작업을 반복하면 간격 맞추기와 복제 작업이 빨라집니다.
  • 같은 도형을 여러 번 만들 때 특히 유용합니다.
  • 정렬과 복제를 함께 사용하면 규칙적인 패턴 만들기에 좋습니다.

복습 워크시트

직접 입력해 보면서 오늘 배운 개념을 다시 정리해 보세요. 아래 버튼으로 채점과 정답 예시 확인이 가능합니다.

빈칸 채우기

1. 실행 취소, 즉 뒤로가기는 이다.

2. 재실행, 즉 앞으로 가기는 이다.

3. 정비율 유지, 다중 선택, 선택 해제 보조에 자주 쓰는 키는 이다.

4. 이전 작업 반복 단축키는 이다.

5. 시작점과 종료점이 같은 패스는 이다.

6. 앵커 포인트, 패스, 핸들을 직접 편집하는 도구는 이다.

점수: 아직 채점 전

서술형 정리

1. 격리모드는 언제 활용하면 좋을까요?

예시 답안: 그룹으로 묶인 개체를 풀지 않고, 그 안의 특정 개체만 따로 수정하고 싶을 때 활용한다.

2. 도형 편집 전에 왜 기본 모양을 먼저 생각해야 할까요?

예시 답안: 원래 어떤 도형에서 시작했는지 알면 어느 포인트를 움직여야 할지 판단하기 쉬워지고, 형태가 무너지지 않게 편집할 수 있다.

3. 오늘 수업에서 내가 특히 기억해야 할 단축키나 개념 3가지를 적어보세요.

예시 답안: Shift로 정비율 유지, Ctrl + D로 반복 작업, 직접선택도구로 앵커 포인트 편집.
