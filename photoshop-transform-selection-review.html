문제지 바로가기

오늘 배운 핵심

  • 자유 변형은 선택한 레이어나 오브젝트의 크기, 회전, 방향, 형태를 바꾸는 기능입니다.
  • 선택 영역은 편집이 적용될 범위를 정하는 기능입니다.
  • 변형과 선택은 따로가 아니라 연결해서 사용합니다.
  • 도구마다 용도가 다르므로 상황에 맞게 골라 쓰는 것이 중요합니다.
포토샵 작업이 꼬일 때는 대부분 선택이 남아 있거나, 다른 레이어를 잡고 있거나, 도구의 옵션을 잘못 보고 있는 경우가 많습니다.

1. 자유 변형 핵심 정리

자유 변형은 선택한 대상의 크기와 방향을 바꾸는 가장 기본적인 조절 기능입니다. 배치, 합성, 크기 수정, 구도 조정에서 거의 항상 사용합니다.

기본 사용

  • Ctrl + T로 자유 변형을 실행합니다.
  • 모서리 핸들을 드래그하면 크기 조절이 됩니다.
  • 바깥쪽으로 커서를 빼면 회전할 수 있습니다.
  • 작업 후에는 Enter로 적용합니다.

보조 키

  • Shift: 정비율 유지
  • Alt: 가운데 기준으로 크기 조절
  • Shift + Alt: 정비율 유지 + 중심 기준 조절
  • 변형 전에는 어떤 레이어가 선택되어 있는지 먼저 확인합니다.

우클릭 메뉴

  • Distort: Ctrl을 활용해 모서리를 독립적으로 당기며 왜곡합니다.
  • Perspective: Ctrl + Alt + Shift를 활용해 원근감을 조절합니다.
  • Warp: 격자를 이용해 부드럽게 휘어지도록 변형합니다.
  • Flip Horizontal: 좌우 반전
  • Flip Vertical: 상하 반전

기억할 포인트

  • Distort는 각 모서리를 따로 조절할 때 적합합니다.
  • Perspective는 멀어지거나 가까워지는 느낌을 줄 때 유용합니다.
  • Warp는 종이, 천, 곡면처럼 부드러운 형태 보정에 적합합니다.
  • Flip 계열은 좌우 대칭이나 방향 변경에 빠르게 씁니다.

2. 선택 영역 기본 기능

선택 영역은 어디를 편집할지 지정하는 기능입니다. 같은 명령이라도 선택이 되어 있느냐 없느냐에 따라 결과가 완전히 달라집니다.

Shift 선택 영역 추가, 다중 선택
Alt 선택 영역 빼기
Shift + Alt 겹치는 부분만 남기기

기본 단축키

  • Ctrl + J: 선택한 부분만 새 레이어로 복제
  • Ctrl + D: 선택 영역 해제
  • 선택 상태를 풀지 않으면 다음 작업이 엉뚱한 부분에 들어갈 수 있습니다.

선택 영역이 중요한 이유

  • 필요한 부분만 정확하게 수정할 수 있습니다.
  • 합성, 색 보정, 변형의 출발점이 됩니다.
  • 선택이 부정확하면 가장자리 품질이 바로 드러납니다.
Selection Tools

3. 선택 도구별 특징

선택 도구는 모양과 선택 방식이 다르기 때문에 대상에 따라 도구를 바꾸는 판단이 중요합니다.

Quick Selection

  • 문지르듯 드래그하면 비슷한 색과 경계를 기준으로 선택합니다.
  • 큰 영역을 빠르게 잡을 때 편리합니다.
  • [: 브러시 축소
  • ]: 브러시 확대
  • Caps Lock: 커서 표시가 달라질 수 있습니다.

Magic Wand

  • 클릭한 색상과 비슷한 영역을 한 번에 선택합니다.
  • Tolerance: 어디까지 비슷한 색으로 볼지 결정합니다.
  • Contiguous 켜기: 연결된 한 구역만 선택
  • Contiguous 끄기: 같은 색 전체 선택

Object Selection

  • 포토샵이 개체를 인식해 선택합니다.
  • 대상을 빠르게 크게 잡을 때 유리합니다.
  • 인물, 사물, 제품처럼 분명한 오브젝트에서 편리합니다.

4. 복습 문제지

빈칸에 직접 정답을 입력하고 채점해보세요. 정답 보기와 초기화, PDF 출력까지 한 번에 가능합니다.

점수: 아직 채점 전

빈칸 채우기

1. 자유 변형 단축키는 이다.
2. 선택 영역 해제 단축키는 이다.
3. 선택 영역만 새 레이어로 복제하는 단축키는 이다.
4. 자유 변형 우클릭 메뉴에서 좌우 반전 기능은 이다.
5. 자유 변형 우클릭 메뉴에서 상하 반전 기능은 이다.
6. 선택 영역 추가는 , 선택 영역 빼기는 이다.
7. 문지르듯 드래그해서 비슷한 색을 고르는 도구는 도구이다.
8. 클릭한 색을 기준으로 비슷한 영역을 한 번에 선택하는 도구는 도구이다.

서술형 작성

1. Distort, Perspective, Warp의 차이를 간단히 설명해보세요.
예시 답: Distort는 모서리를 독립적으로 왜곡하고, Perspective는 원근감을 주며, Warp는 부드럽게 휘어지도록 변형한다.
2. Shift 키가 변형 작업과 선택 작업에서 각각 어떤 역할을 하는지 적어보세요.
예시 답: 변형에서는 정비율 유지, 선택에서는 선택 영역 추가에 사용한다.
3. Alt 키가 변형 작업과 선택 작업에서 각각 어떤 역할을 하는지 적어보세요.
예시 답: 변형에서는 가운데 기준 크기 조절, 선택에서는 선택 영역 빼기에 사용한다.
4. Magic Wand의 Tolerance와 Contiguous가 무엇을 조절하는지 적어보세요.
예시 답: Tolerance는 비슷한 색의 허용 범위를 정하고, Contiguous는 연결된 영역만 선택할지 같은 색 전체를 선택할지 정한다.
5. Object Selection 도구의 장점을 적어보세요.
예시 답: 포토샵이 개체를 인식해 대상을 빠르게 잡아주므로 큰 형태를 선택할 때 효율적이다.

5. 정답 확인 요약

채점 기능은 객관식과 빈칸형에 적용됩니다. 서술형은 아래 예시 답안과 비교해서 스스로 확인하면 됩니다.

핵심 정답
  • 자유 변형: Ctrl + T
  • 선택 해제: Ctrl + D
  • 선택 영역 복제: Ctrl + J
  • 좌우 반전: Flip Horizontal
  • 상하 반전: Flip Vertical
도구 구분
  • Quick Selection: 문질러서 비슷한 색과 경계 선택
  • Magic Wand: 클릭한 색상 기준 선택
  • Object Selection: 개체 인식 기반 선택
  • Shift: 정비율 유지 / 선택 추가
  • Alt: 중심 기준 조절 / 선택 빼기
한 줄 정리

선택 영역은 어디를 바꿀지 정하는 기능, 변형은 어떻게 바꿀지 정하는 기능입니다. 이 둘을 연결해서 쓸 수 있어야 포토샵 기본 작업 흐름이 잡힙니다.
