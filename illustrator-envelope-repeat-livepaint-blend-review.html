1. 오늘 복습의 큰 흐름

이번 단원은 “오브젝트를 변형하고, 묶고, 칠하고, 연결하는 과정”으로 보면 훨씬 정리가 쉽다.

수업 전체 지도
4 왜곡 파트: Make with Warp, Make with Mesh, Make with Top Object, Expand
3 반복 방식: Radial, Grid, Mirror
2 라이브페인트 핵심: 페인트통으로 칠하고 필요 시 Expand
1 블렌드는 두 개 이상의 개체를 섞어 색과 모양이 함께 변화되도록 만든다.
암기 포인트: 왜곡은 형태를 바꾸는 기능, 반복은 자동 배열 기능, 라이브페인트는 닫힌 면처럼 칠하는 기능, 블렌드는 개체 사이를 연결하는 기능이다.

2. 왜곡

Object → Envelope Distort에서 형태를 강하게 변형한다. 결과물을 확정할 때는 Expand를 기억한다.

잉<왜곡>
기능 경로

Envelope Distort

  • 1
    Object → Envelope Distort → Make with Warp
    미리 준비된 휨 형태로 빠르게 왜곡할 때 사용한다.
  • 2
    Make with Mesh
    행과 열을 가진 망으로 직접 눌러가며 세밀하게 형태를 조절한다.
  • 3
    Make with Top Object
    맨 위에 있는 오브젝트의 형태를 기준으로 아래 오브젝트를 감싸듯 왜곡한다.
  • 4
    Expand
    왜곡된 상태를 일반 패스로 확정하고 다음 작업으로 넘긴다.
언제 쓰나

상황별 구분

빠른 글자 왜곡
제목이나 로고를 빠르게 휘게 만들고 싶다면 Warp가 가장 빠르다.
면을 직접 누르기
형태가 복잡하고 손으로 잡아당기듯 조정하고 싶다면 Mesh가 유리하다.
기준 모양 따라가기
물결, 띠, 리본, 아이스크림 윤곽 같은 특정 외곽을 따라가려면 Top Object를 쓴다.

원본 텍스트나 도형이 아직 수정될 수 있다면 복제본을 하나 남겨 두는 편이 안전하다.

왜곡 후

수정이 끝난 뒤에는 Expand로 확정해야 패스파인더, 색 분할, 추가 편집이 안정적으로 진행된다.

3. 아이스크림 그래픽 실습 흐름

단축키를 따로 외우기보다, 한 장면을 만드는 순서로 묶으면 훨씬 오래 기억된다.

실습 루틴
텍스트 윤곽선 만들기 컴파운드 패스 뒤로 제자리 복제 Unite Offset Path 맨 위로 올리기
문자 처리

윤곽선 만들기

Ctrl + Shift + O

글자를 더 이상 텍스트로 두지 않고 패스로 바꾸는 작업이다. 수업에서는 “글자를 깬다”라고 설명한다.

패스 정리

컴파운드 패스

Ctrl + 8

안쪽 구멍이 있는 글자나 여러 조각을 하나의 묶음처럼 다루고 싶을 때 사용한다.

복제

뒤로 제자리 복제

Ctrl + C / Ctrl + B

같은 자리에 복사한 뒤 뒤쪽으로 보내기 때문에 외곽선, 그림자, 배경 판을 만들 때 자주 쓴다.

합치기

Pathfinder Unite

Pathfinder → Unite

겹친 도형을 하나의 큰 덩어리로 합친다. 아이스크림 바탕 모양을 정리할 때 유용하다.

두께 만들기

Offset Path

원형을 바깥쪽이나 안쪽으로 일정 거리만큼 확장하거나 축소한다. 로고 테두리, 스티커 외곽선, 아이스크림 외피를 만들 때 매우 자주 등장한다.

  • 복제본에 적용하면 원본을 건드리지 않고 두께를 추가할 수 있다.
  • 값이 너무 크면 모서리가 깨져 보일 수 있으니 미리보기로 확인한다.
레이어 순서

맨 위로 보내기

Ctrl + Shift + ]

장식 오브젝트나 글자를 가장 위에 다시 올릴 때 쓰는 정리용 단축키다. 완성도는 도형 자체보다도 쌓이는 순서에서 많이 갈린다.

실습에서 흔한 실수: 외곽선 도형을 만들고도 맨 위로 올리지 않아 글자가 가려지는 경우가 많다.

4. 반복

Object → Repeat는 한 개의 오브젝트를 규칙적으로 퍼뜨리는 자동 배열 도구다.

자동 배열
Radial

방사형 반복

원형으로 둘러가며 배치된다. 꽃잎, 시계, 빛, 원형 로고 장식에 적합하다.

Grid

격자 반복

가로 세로로 규칙적인 배열을 만든다. 패턴 초안이나 아이콘 진열, 포스터 배경에 잘 맞는다.

Mirror

대칭 반복

한쪽을 수정하면 반대쪽이 함께 반영된다. 얼굴, 날개, 장식 문양처럼 좌우 대칭 작업에 편하다.

Repeat는 “복제”와 다르다. 단순히 여러 개를 찍어내는 것이 아니라, 원본과 연결된 상태로 반복 구조를 유지한다는 점이 핵심이다.

5. 라이브페인트와 원근감 격자

면을 칠하는 방식과 화면 보조 도구를 함께 묶어 두면 실무에서 혼동이 줄어든다.

칠하기 + 보조선
Live Paint

라이브페인트

  • 1
    오브젝트를 선택한 뒤 라이브페인트 그룹으로 만든다.
  • 2
    페인트통 도구로 면 단위로 색을 넣는다.
  • 3
    결과를 일반 패스로 정리해야 할 때 Expand를 적용한다.

닫혀 있지 않은 선처럼 보여도 칠하기 편하게 면을 인식해 주는 것이 장점이다.

Perspective Grid

원근감 격자

원근 작업을 돕는 보조선 도구다. 필요 없을 때는 빠르게 끄는 습관이 중요하다.

  • Ctrl + Shift + I : 원근감 격자 켜기 / 끄기
  • 갑자기 화면에 파란 격자가 떠서 작업이 불편하면 이 단축키를 먼저 떠올린다.

6. 블렌드

두 개 이상의 개체를 섞어 중간 단계를 자동으로 만들어 주는 기능이다. 색과 모양이 함께 변화된다.

연결과 변화
실행

기본 단축키

  • 1
    두 개 이상의 오브젝트를 준비한다.
  • 2
    Ctrl + Alt + B로 블렌드를 만든다.
  • 3
    W를 누른 뒤 Enter로 블렌드 옵션을 연다.
화면 정리

바운딩 박스 표시 전환

Ctrl + Shift + B로 바운딩 박스를 숨기거나 다시 보이게 할 수 있다. 블렌드 작업 중 선택 상자가 거슬릴 때 유용하다.

수업 메모: W / Enter 조합은 블렌드 옵션을 빠르게 다시 확인할 때 자주 쓴다.

7. 복습문제

선택형과 서술형을 섞었다. 답을 고르거나 입력한 뒤 바로 채점해 보자.

즉시 채점 가능
문제 1

Object → Envelope Distort에서 맨 위 오브젝트의 형태를 기준으로 왜곡하는 명령은 무엇인가?

문제 2

왜곡이나 라이브페인트 결과를 일반 패스로 확정할 때 공통으로 떠올려야 하는 명령을 입력하시오.

문제 3

아이스크림 그래픽 작업 순서로 가장 알맞은 것을 고르시오.

문제 4

Object → Repeat에 포함되는 방식을 모두 고르시오.

문제 5

원근감 격자를 끄는 단축키를 고르시오.

문제 6

블렌드 기능을 한 문장으로 정리해 보시오.

힌트: “두 개 이상의 개체”, “섞는다”, “색과 모양 변화” 중 2개 이상을 포함해 보자.

서술형은 핵심어 포함 여부를 기준으로 안내한다.
점수: 0 / 5
객관식 5문항은 자동 채점, 서술형 1문항은 핵심어 포함 여부를 안내한다.

복습 팁
시험이나 실습평가에서는 메뉴 이름만 쓰지 말고, “어떤 상황에서 왜 쓰는가”까지 같이 떠올려야 작업 속도가 빨라진다. 특히 Expand, Ctrl + Shift + O, Ctrl + Alt + B, Ctrl + Shift + I는 자주 다시 나온다.
