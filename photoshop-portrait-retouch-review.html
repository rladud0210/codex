1. 인물보정 전체 순서

실습에서 중요한 것은 기능 이름을 외우는 것보다, 어떤 순서로 써야 결과가 자연스러운지 이해하는 것입니다.

1

Spot Healing Brush

작은 트러블, 점, 잔주름처럼 빠르게 지울 수 있는 부분을 먼저 정리합니다.

2

Healing Brush

결이나 톤을 참고해서 자연스럽게 복원해야 하는 부분은 Healing Brush로 정리합니다.

3

Path Tool

윤곽 정리나 정교한 선택이 필요할 때 펜 경로를 잡아 작업 범위를 정확히 만듭니다.

4

Smart Object + Smart Blur

레이어를 스마트 오브젝트로 바꾼 뒤 Filter → Smart Blur를 적용해 피부를 매끈하게 정리할 준비를 합니다.

5

Smart Filter Mask 복구

스마트 필터 마스크를 검은색으로 채워 숨기고, 흰색 브러시로 필요한 피부에만 다시 드러냅니다.

6

Liquify + 색 보정

마지막에 얼굴선이나 비율을 미세하게 정리하고, 입술색과 전체 톤을 보정해 마무리합니다.

2. 도구별 역할 정리

비슷해 보이는 도구라도 역할이 다릅니다. 도구 선택이 맞아야 피부결이 덜 무너집니다.

기초 정리

Spot Healing Brush

클릭만으로 처리하기 쉬운 작은 잡티 제거에 적합합니다. 빠르지만 넓은 영역을 과하게 문지르면 결과가 뭉개질 수 있습니다.

짧게, 작게, 필요한 만큼만 사용하는 것이 핵심입니다.
정교 복원

Healing Brush

샘플 지점을 직접 잡아 톤과 질감을 맞추기 때문에 코 주변, 눈 밑, 굴곡이 있는 피부 보정에 더 안정적입니다.

Alt로 샘플을 잡고, 피부 방향을 따라 짧게 복원합니다.
정확한 선택

Path Tool

경계를 깔끔하게 잡아야 하는 턱선, 입술 라인, 특정 오브젝트 외곽 정리에 유리합니다.

패스를 잡아두면 나중에 선택 영역으로 다시 불러올 수 있습니다.
비파괴 보정

Smart Object

필터를 다시 수정할 수 있게 만드는 기반 단계입니다. 피부 보정 강도를 나중에 조절할 수 있어서 실무에서 안전합니다.

원본을 직접 망가뜨리지 않고, 필터값을 재편집할 수 있습니다.
부분 적용

Smart Filter Mask

스마트 블러를 전체에 건 뒤 필요한 부위에만 다시 보이게 만드는 핵심 장치입니다.

검은색은 숨김, 흰색은 보임. 피부에만 흰색 브러시를 칠합니다.
최종 정리

Liquify / Tone Correction

얼굴형, 턱선, 입술 라인처럼 형태를 미세하게 정리한 뒤 입술색, 밝기, 전체 톤을 손봐 마무리합니다.

리퀴파이는 과하면 티가 나므로 마지막에 아주 약하게 쓰는 편이 안전합니다.

3. 스마트 필터 피부 보정 핵심

이번 실습의 핵심은 블러를 무조건 올리는 것이 아니라, 마스크로 필요한 피부만 골라 쓰는 것입니다.

실습 순서

Smart Blur 적용 후 피부만 복구하기

  1. 보정 레이어를 스마트 오브젝트로 변환합니다.
  2. Filter → Smart Blur를 적용합니다.
  3. Smart Filters 마스크를 선택합니다.
  4. 전경색을 검은색으로 두고 전체 효과를 숨깁니다.
  5. 흰색 브러시로 이마, 볼, 턱 등 필요한 피부만 복구합니다.
  6. 눈, 코, 입, 머리카락, 윤곽선은 지나치게 칠하지 않도록 주의합니다.
왜 이 방식인가

전체가 아니라 피부만 매끈하게

1
피부결만 정리

눈썹, 속눈썹, 입술 경계까지 같이 흐려지는 것을 막을 수 있습니다.

2
강도 재조절 가능

스마트 필터라서 필터값도 바꾸고, 마스크 범위도 다시 칠할 수 있습니다.

3
자연스러운 결과

필요한 부위만 보정하므로 피부는 정리되지만 얼굴 입체감은 남길 수 있습니다.

4. 수업에서 꼭 체크할 포인트

학생들이 가장 많이 실수하는 지점을 작업 전후로 확인할 수 있게 정리했습니다.

실수 방지

피부 보정할 때 자주 생기는 문제

!
블러를 너무 강하게 주기

플라스틱처럼 보여서 인물 사진이 부자연스러워집니다.

!
눈, 코, 입까지 같이 복구하기

선명해야 할 경계가 흐려지면 사진이 바로 답답해집니다.

!
잡티 제거를 과하게 반복하기

피부결이 사라져서 실제보다 인공적인 결과가 나옵니다.

마무리 기준

좋은 인물보정의 기준

✓
정리됐지만 원래 피부처럼 보이기

잡티는 줄었는데 피부결과 명암은 남아 있어야 합니다.

✓
리퀴파이가 티 나지 않기

얼굴형은 정돈되지만 배경선이 휘거나 왜곡되면 안 됩니다.

✓
입술색과 톤이 전체 사진과 어울리기

입술만 튀지 않고 전체 인물 분위기 안에서 정리되어야 합니다.

6. 복습 문제

기능 이름과 작업 의도를 함께 점검하는 문제입니다. 선택 또는 입력 후 바로 확인할 수 있습니다.

문제 1 · 객관식

Healing Brush를 Spot Healing Brush보다 우선적으로 써야 하는 경우는?

문제 2 · 객관식

스마트 오브젝트로 변환하는 가장 큰 이유는?

문제 3 · 객관식

스마트 필터 마스크에서 검은색은 무엇을 뜻하나요?

문제 4 · 서술형

리퀴파이는 왜 마지막 단계에 두는 편이 좋을까요?

현재 학습 진행 0 / 6 완료 워크시트 2개와 복습 문제 4개를 확인하면 점수가 갱신됩니다.
기억할 핵심 문장: 작은 잡티는 먼저 정리하고, 피부 보정은 스마트 필터 마스크로 필요한 피부에만 적용한 뒤, 마지막에 리퀴파이와 색 보정으로 마무리한다.
