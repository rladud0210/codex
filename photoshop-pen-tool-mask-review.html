  • 곡선은 많이 찍는 것이 아니라, 정확한 지점에 최소한으로 찍는 것이 핵심입니다.

펜툴 핵심 원리

펜툴은 손 감각보다 구조 이해가 먼저입니다. 아래 네 가지 원칙을 기준으로 그리면 훨씬 안정적으로 선이 잡힙니다.

곡선을 곡선처럼 쫓아가기보다, 곡률이 시작되는 지점과 끝나는 지점을 직선 기준으로 먼저 찾는 것이 중요합니다.
1

앵커포인트는 정확한 위치에

곡선 전체를 따라가려 하지 말고, 곡률이 바뀌는 시작점과 끝점을 먼저 찾습니다.

  • 처음에는 곡선이 아니라 직선이라고 생각해 보기
  • 곡률이 시작하는 부분과 끝나는 부분 관찰하기
2

곡선을 그리는 중에는 핸들을 끊지 않기

핸들을 자주 끊으면 흐름이 깨져서 곡선이 각져 보이기 쉽습니다. 먼저 부드러운 연결을 우선합니다.

3

앵커포인트 개수는 최소한으로

포인트를 많이 찍는다고 더 정교해지지 않습니다. 적은 수로 큰 흐름을 잡는 쪽이 수정도 쉽습니다.

4

방향선은 진행 방향으로

방향선은 지금 그리는 지점이 아니라 다음 곡선을 준비하는 도구입니다. 항상 앞으로 뻗는다는 감각으로 봅니다.

펜툴 편집 요약

그리는 동작과 수정 동작을 따로 기억하지 말고, 클릭과 보조키 조합으로 묶어서 외우는 편이 빠릅니다.

그리기

곡선 그리기

클릭, 드래그로 앵커포인트와 방향선을 동시에 만듭니다.

그리기

직선 그리기

클릭, 클릭으로 직선 구간을 연결합니다.

마무리

다 그렸을 때

Enter를 눌러 현재 패스 입력을 마무리합니다.

편집

핸들 끊기

Alt를 누른 상태로 클릭하면 방향선을 끊어 다음 구간 흐름을 분리할 수 있습니다.

편집

그리는 중 앵커포인트 수정

Ctrl을 누르고 드래그하면 Direct Selection처럼 포인트와 핸들을 임시 수정할 수 있습니다.

편집

핸들 다시 그리기

Alt + 드래그는 Convert Point Tool처럼 작동하여 방향선을 새로 잡을 때 사용합니다.

Shape 와 Path 구분

수업 중 자주 헷갈리는 부분입니다. 작업 목적을 먼저 정하면 툴 옵션 선택이 훨씬 쉬워집니다.

Shape

모양을 그릴 거다

결과물이 바로 도형 레이어로 만들어집니다. 색이 들어간 형태를 만들거나 디자인 요소를 제작할 때 적합합니다.

도형 레이어 생성 색상/스트로크 적용 디자인 요소 제작
Path

선택영역을 잡을 거다

벡터 경로만 남겨두고 나중에 선택영역, 마스크, 합성 작업으로 이어갑니다. 실무에서는 누끼와 정교한 선택에 자주 씁니다.

선택영역 변환 가능 마스크 작업 연결 합성용 경로 활용
핵심 단축

패스를 선택영역으로 바꾸기

Ctrl + Enter를 누르면 현재 패스를 선택영역으로 변환할 수 있습니다. 누끼 작업과 레이어 마스크 연결의 출발점이 되는 조합입니다.

클리핑 마스크 / 레이어 마스크

둘 다 가리는 기능처럼 보이지만 역할이 다릅니다. 하나는 “끼워 넣기”, 다른 하나는 “비파괴적으로 지우기”입니다.

Clipping Mask

액자 안에 그림 끼워 넣기

아래 레이어의 모양을 기준으로 위 레이어를 보이게 만드는 기능입니다.

  • 레이어와 레이어 사이에 마우스를 두고 Alt + 클릭
  • “액자” 레이어 바로 위에 “그림” 레이어가 있어야 함
  • 여러 액자에 한 번에 적용하려면 액자를 그룹으로 묶고 그룹에 클리핑 마스크 적용
Layer Mask

원본을 비파괴적으로 가리기

지우개처럼 없애지만, 실제 픽셀을 삭제하지 않아서 언제든 복구할 수 있습니다.

  • 검은색: 가려짐
  • 회색: 반투명
  • 흰색: 보임, 복구됨
레이어 마스크 팁

Shift 클릭

마스크 썸네일을 Shift + 클릭하면 마스크를 임시로 활성화/비활성화할 수 있습니다.

레이어 마스크 팁

검은색 마스크 바로 만들기

Alt를 누른 상태로 레이어 마스크를 적용하면 처음부터 전체가 가려진 검은색 마스크가 생성됩니다.

레이어 마스크 팁

선택영역으로 배경 제거

선택영역을 만든 뒤 레이어 마스크를 적용하면 선택영역 바깥은 가려져서 배경 제거처럼 사용할 수 있습니다.

복습 워크시트

읽고 끝내지 말고 직접 답해 보세요. 객관식은 채점되고, 서술형은 예시 답안을 열어 비교할 수 있습니다.

1. 곡선이 시작되는 부분과 끝나는 부분을 보고 앵커포인트를 두는 이유로 가장 적절한 것은?
2. 펜툴로 곡선을 그리는 기본 동작은 무엇인가?
3. 다음 중 패스를 선택영역으로 바꾸는 단축키는?
4. 레이어 마스크에서 흰색이 의미하는 것은?
5. 빈칸 채우기: 여러 개의 액자에 그림을 끼워 넣고 싶다면 액자를 __________ 으로 묶고, 그 그룹에 클리핑 마스크를 적용한다.
6. 서술형: Shape 와 Path 의 차이를 2문장 안으로 설명해 보세요.
Shape는 결과물이 도형 레이어로 바로 만들어져 모양 자체를 디자인할 때 사용한다. Path는 벡터 경로만 만들고, 나중에 선택영역이나 마스크로 변환해 합성 작업에 활용한다.
점수: 아직 채점 전
객관식 4문항과 빈칸 1문항을 채점합니다. 서술형은 예시 답안과 비교해 스스로 점검하세요.

서술형은 자동 채점하지 않습니다. 대신 예시 답안을 열어 핵심 키워드가 들어갔는지 스스로 확인하도록 구성했습니다.
