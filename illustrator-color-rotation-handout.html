Shortcut Review

자주 쓰는 레이어 · 보기 단축키

겹쳐진 오브젝트의 앞뒤 순서를 빠르게 바꾸는 단축키는 선택 도구 작업과 함께 반복적으로 사용됩니다.

Ctrl + Y

윤곽선 모드

복잡한 일러스트에서 실제 면과 효과 대신 선 구조만 확인할 때 사용합니다.

Ctrl + ]

한 칸만 위로

선택한 오브젝트를 바로 위의 오브젝트 한 단계 앞으로 이동합니다.

Ctrl + Shift + ]

맨 위로

선택한 오브젝트를 현재 레이어의 가장 앞쪽으로 보냅니다.

Ctrl + [

한 칸만 아래로

선택한 오브젝트를 바로 아래의 오브젝트 한 단계 뒤로 이동합니다.

Ctrl + Shift + [

맨 뒤로

선택한 오브젝트를 현재 레이어의 가장 뒤쪽으로 보냅니다.

체크 포인트

앞뒤 순서 감각 익히기

한 칸 이동과 맨 앞/맨 뒤 이동을 구분해서 기억해야 작업 속도가 빨라집니다.

Color Tools

컬러 관련 핵심 기능

컬러는 패널 이름만 외우는 것이 아니라 Fill / Stroke 순서와 스포이드 옵션까지 함께 이해해야 실수가 줄어듭니다.

1. 컬러 피커

색상을 직접 선택하고 수치로 조절할 수 있는 기본 창입니다.

2. 컬러 패널

RGB, CMYK 같은 수치 기반으로 색을 세밀하게 조절할 때 사용합니다.

3. 스와치

자주 쓰는 색을 저장해 두고 반복 적용할 때 가장 편리한 패널입니다.

중요: Fill Color와 Stroke Color는 앞에 보이는 쪽이 현재 적용 대상입니다. 순서의 영향을 받기 때문에, 원하는 대상이 앞에 올라와 있는지 먼저 확인해야 합니다.
4. 스포이드

Eyedropper 핵심

  • 기본 사용 시 색만이 아니라 모든 속성이 함께 추출될 수 있습니다.
  • 색만 뽑아오고 싶다면 Shift를 함께 사용합니다.
  • 반대로 적용시키고 싶다면 Alt를 사용합니다.
시험 포인트

실수하기 쉬운 부분

  • 스포이드가 색만 가져오는 도구라고 단정하면 오답이 됩니다.
  • Shift 기능은 별표 표시할 정도로 중요합니다.
  • Fill / Stroke의 앞뒤 상태를 확인하지 않으면 원하는 곳에 색이 안 들어갈 수 있습니다.
Gradient Review

그라데이션 = 그라디언트

용어와 도구, 종류를 함께 연결해서 기억하면 문제 풀이와 실습 둘 다 수월합니다.

Linear

선형

한 방향으로 부드럽게 이어지는 기본 그라디언트입니다.

Radial

방사형

중심에서 바깥으로 퍼지는 원형 계열 그라디언트입니다.

Freeform

자유형

여러 지점에 색을 직접 배치하며 자유롭게 흐름을 만드는 방식입니다.

항목 기억할 내용 관련 단축키 / 메뉴
그라데이션 실행 점(.) 키로 빠르게 실행할 수 있습니다. .
그라디언트 툴 방향과 범위를 직접 조절할 때 사용합니다. G
색 재구성 여러 색 조합을 한 번에 바꿀 때 사용합니다. Edit → Edit Colors → Recolor Artwork
Transform

변형: 회전은 중심축 기준

회전은 단순히 돌리는 기능이 아니라 중심축을 어디에 두느냐가 핵심입니다.

회전 도구 선택

R 키로 회전 도구를 선택합니다.

중심축 클릭 후 돌리기

회전 중심이 될 지점을 먼저 정한 뒤 오브젝트를 회전합니다.

옵션창 열기

Enter를 누르면 Rotate 옵션창이 열려 수치 입력이 가능합니다.

복제 회전의 핵심

Alt + 클릭으로 중심축을 지정하면 복제와 반복 작업에 매우 유리합니다.

반복

Ctrl + D

직전 변형을 반복 적용합니다. 복제 회전 후 패턴처럼 연속 배치할 때 자주 사용합니다.

수치 입력

마이너스와 수식

  • 마이너스 수치를 입력하면 반대 방향으로 회전합니다.
  • 수식을 그대로 입력해도 자동 계산됩니다.
  • 예: 360/3, 360*2, 360+3
암기 포인트: 회전은 항상 중심축 기준으로 생각합니다. 중심축 지정, Alt + 클릭, Ctrl + D 조합을 묶어서 기억해 두면 좋습니다.
