기준 개체 정렬

핵심 순서는 짧습니다. 먼저 모두 선택하고, 기준으로 만들고 싶은 개체를 한 번 더 클릭합니다.

1

다중 선택

정렬할 개체를 모두 선택합니다. 선택 도구 단축키는 V입니다.

2

한 번 더 클릭

기준으로 만들고 싶은 개체를 한 번 더 클릭합니다. 더 진한 선택 테두리가 보이면 기준 개체입니다.

3

정렬 실행

Align 패널에서 원하는 정렬 버튼을 누릅니다. 기준 개체는 고정되고 나머지만 움직입니다.

기억 문장: 전체 선택 → 기준 개체 한 번 더 클릭 → 정렬 버튼.

분포 간격

간격 수치를 정확히 넣어 정리하려면 기준 개체가 필요합니다.

  1. 간격을 맞출 여러 개체를 다중 선택합니다.
  2. 움직이지 않을 기준 개체를 한 번 더 클릭합니다.
  3. Align 패널의 Distribute Spacing 영역에서 간격 수치를 입력합니다.
  4. 가로 간격이면 Horizontal Distribute Space, 세로 간격이면 Vertical Distribute Space를 누릅니다.
중요: 분포 간격에 수치를 입력해서 쓰는 기능은 기준 개체가 있어야 제대로 사용할 수 있습니다.

반전 · 변형 · 잠금

정렬 전에 건드리면 안 되는 개체는 잠그고, 반복 형태는 반전과 변형 도구를 함께 확인합니다.

그룹 정렬

그룹을 어떻게 묶을까?

여러 개의 개체 정렬을 맞출 때는 의미 단위로 먼저 그룹을 묶습니다. 같이 움직여야 하는 요소끼리 묶고, 따로 정렬해야 하는 요소는 분리합니다.

변형 도구

반전과 변형

반전 도구와 회전, 크기 조절 같은 변형 도구는 기준점이 중요합니다. 결과가 이상하면 기준점 위치를 먼저 확인합니다.

잠금

고정할 개체 보호

Ctrl 2로 잠금, Ctrl Alt 2로 잠금 해제합니다. 배경이나 기준 요소를 실수로 선택하지 않게 할 때 유용합니다.

체크리스트

작업 전후로 아래 항목을 확인합니다.

  • 여러 개의 개체 정렬을 맞출 때 그룹으로 묶을지 먼저 판단했다.
  • 그룹은 같이 움직여야 하는 개체끼리 묶었다.
  • 다중 선택 상태에서 기준 개체를 한 번 더 클릭했다.
  • 분포 간격 수치를 쓰기 전에 기준 개체가 지정되어 있는지 확인했다.
  • 정렬 전에 같이 움직여야 하는 개체와 따로 움직여야 하는 개체를 구분했다.
  • 반전 도구와 변형 도구의 기준점 위치를 확인했다.
  • 건드리면 안 되는 개체는 Ctrl 2로 잠갔다.
  • 잠금 해제는 Ctrl Alt 2로 할 수 있다.

복습 문제

빈칸은 자동 채점하고, 서술형은 정답 보기로 예시 답안을 확인합니다.

빈칸 문제

1. 선택 도구의 단축키는?

2. 다중 선택 후 기준 개체를 지정하려면 기준으로 만들 개체를 어떻게 해야 하는가?

3. 분포 간격에 수치를 넣어 쓰려면 무엇이 있어야 하는가?

4. 잠금 단축키는?

5. 잠금 해제 단축키는?

점수: 아직 채점 전

서술형 문제

1. 여러 개체를 그룹으로 묶은 후 정렬해야 하는 상황을 설명해보세요.

예시 답: 아이콘과 텍스트가 하나의 버튼처럼 같이 움직여야 할 때, 아이콘과 텍스트를 먼저 그룹으로 묶은 뒤 다른 요소와 정렬한다.

2. 기준 개체 정렬 결과가 이상할 때 확인할 것 3가지를 적어보세요.

예시 답: 기준 개체를 한 번 더 클릭했는지, Align to Key Object 상태인지, 아트보드 기준 정렬이 켜져 있지 않은지 확인한다.

3. 정렬 전에 잠금 기능을 사용하면 좋은 상황을 적어보세요.

예시 답: 배경이나 이미 맞춰 둔 기준 요소처럼 움직이면 안 되는 개체가 있을 때 Ctrl 2로 잠근 뒤 나머지 개체를 정렬한다.
오늘의 핵심: 기준 개체는 다중 선택 후 한 번 더 클릭해서 지정합니다. 분포 간격 수치 입력은 기준 개체가 있어야 안정적으로 사용할 수 있고, 정렬 전에는 그룹과 잠금 상태를 먼저 확인합니다.
