Illustrator Review Sheet

펜툴 복습자료

앵커포인트 위치를 정확하게 보고, 핸들 흐름을 끊지 않고, 최소한의 점으로 패스를 정리하는 연습용 자료입니다. 아래 복습문제는 직접 선택하거나 입력한 뒤 채점할 수 있습니다.

핵심 원칙 펜툴 편집 펜툴 보조 선 · 패스 편집 복습문제 정답 확인
복습문제 풀기
위치 곡률이 시작하고 끝나는 지점에 앵커포인트를 둡니다.
흐름 곡선 진행 중에는 핸들을 함부로 끊지 않습니다.
개수 앵커포인트는 최소한으로 사용해야 수정이 쉽습니다.
방향선 방향선은 항상 진행방향으로 뻗는다고 생각합니다.
직선으로 구조를 먼저 보고, 필요한 지점에서만 곡선으로 전환하면 펜툴이 훨씬 안정됩니다.

핵심 원칙

펜툴을 잘 쓰는 기준은 많이 찍는 것이 아니라 정확하게 찍는 것입니다. 아래 네 가지를 먼저 몸에 익히면 패스가 안정됩니다.

기본 감각 정리
원칙 1

앵커포인트를 정확한 위치에 찍는다

  • 처음부터 곡선으로 보지 말고 직선 구조로 나눠서 생각합니다.
  • 곡률이 시작하는 부분과 끝나는 부분을 먼저 찾습니다.
  • 중간마다 촘촘하게 점을 찍는 습관은 줄입니다.
원칙 2

곡선을 그리는 중에는 핸들을 끊지 않는다

  • 부드럽게 이어지는 곡선은 한 흐름으로 보고 핸들을 유지합니다.
  • 중간에 핸들을 자주 끊으면 곡선 결이 꺾여 보이기 쉽습니다.
원칙 3

앵커포인트 개수는 최소한으로 쓴다

  • 점이 많을수록 수정 지점이 늘어나고 균형 잡기가 어려워집니다.
  • 같은 형태를 더 적은 점으로 만들 수 있는지 항상 다시 봅니다.
원칙 4

방향선은 항상 진행방향으로 뻗는다

  • 지금 그리고 있는 흐름이 어디로 가는지 생각하며 드래그합니다.
  • 방향선을 반대로 의식하면 다음 곡선 연결이 거칠어집니다.
빠른 한 줄 기억 직선으로 구조를 나누고, 꼭 필요한 위치에만 앵커포인트를 두고, 곡선 흐름은 핸들로 이어갑니다.

펜툴 편집

그리는 중간에 어떤 조작을 해야 하는지 정확히 기억해야 실제 작업 속도가 올라갑니다.

실전 조작
그리기 기본

클릭과 드래그 구분

  • 곡선 그리기: 클릭 후 드래그
  • 직선 그리기: 클릭 후 클릭
  • 다 그렸다면 Enter로 현재 그리기를 마칩니다.
핸들 제어

핸들 끊기와 다시 그리기

  • 핸들 끊기: 직전에 만든 클릭 앵커포인트를 다시 클릭
  • 핸들 다시 그리기: Alt를 누른 상태로 드래그
수정

그리는 중 앵커포인트와 핸들 수정

  • Ctrl을 누르고 앵커포인트나 핸들을 직접 수정합니다.
  • 완성 후 수정만이 아니라 그리는 중간 수정도 익숙해져야 합니다.
작업 습관

끊어야 할 때와 이어야 할 때를 구분

  • 부드러운 곡선 구간은 흐름을 이어갑니다.
  • 방향이 급격히 바뀌는 모서리 구간에서만 핸들 끊기를 판단합니다.

펜툴 보조 도구

펜툴만이 아니라 관련 보조 도구의 역할까지 정리해 두면 패스 편집이 훨씬 편해집니다.

보조 도구 묶음
Curvature Tool

곡선을 더 쉽게 잡을 때

  • 클릭만 해도 곡선을 만들기 쉬운 도구입니다.
  • Alt 클릭 또는 더블클릭으로 직선/곡선 전환을 보조할 수 있습니다.
  • 핸들을 세밀하게 직접 조절하는 방식은 어렵습니다.
Add Anchor Point

앵커포인트 추가

  • 패스 위를 정확하게 클릭해야 원하는 위치에 점이 추가됩니다.
  • 무작정 점을 늘리는 용도가 아니라 꼭 필요한 수정 지점을 만드는 용도입니다.
Delete Anchor Point

앵커포인트 삭제

  • 앵커포인트 위를 정확하게 클릭하면 점이 삭제됩니다.
  • 닫힌 패스를 유지한 채로 점 수를 줄일 수 있습니다.
Anchor Point Tool

직선과 곡선을 전환

  • 직선에서 곡선으로: 드래그
  • 곡선에서 직선으로: 클릭
  • 핸들 추가와 핸들 삭제를 담당하는 핵심 편집 도구입니다.

선 그리기 · 선 편집 · 패스 편집

펜툴 복습과 함께 자주 같이 쓰는 선 관련 도구와 패스 메뉴도 한 번에 정리합니다.

함께 기억할 기능
선 그리기

기본 선 도구

  • Line Segment Tool
  • Rectangular Grid Tool
선 편집

선 굵기와 형태 조절

  • Stroke 패널에서 선의 굵기와 스타일을 관리합니다.
  • Width Tool로 구간별 선 굵기 변화를 줄 수 있습니다.
패스 편집

Object > Path

  • 패스 관련 편집 기능을 묶어 관리하는 메뉴입니다.
  • Offset Path처럼 여백을 일정하게 늘리거나 줄이는 기능을 자주 씁니다.
Offset Path

같은 간격으로 확장하거나 축소

  • 상하좌우 여백을 같은 크기로 늘리거나 줄일 수 있습니다.
  • 자동 복제처럼 일정 간격 버전을 빠르게 만들 때 유용합니다.

  • 곡선도 처음에는 직선 구조로 나눠서 본다.
