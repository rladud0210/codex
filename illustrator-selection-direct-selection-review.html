선택 툴과 직접 선택 툴, 아트보드까지 한 번에 복습

이번 수업 메모를 기준으로 정리한 복습 자료입니다. 선택 도구와 직접 선택 도구의 차이, 아트보드 조절, 자주 쓰는 단축키, 패스 구조를 같이 묶어 실제 작업 흐름으로 다시 익히는 데 초점을 맞췄습니다.

아트보드 선택 툴 직접 선택 패스 구조 복습문제 정답

아트보드 먼저 정리

Ctrl + N으로 새 문서를 만든 뒤 가장 먼저 확인할 기본입니다. 아트보드는 작업 화면 자체라서, 크기와 위치, 배열 순서를 정리해야 이후 선택 작업도 덜 꼬입니다.

작업 화면 관리
아트보드 툴

크기, 위치, 재정렬

  • 아트보드 툴로 아트보드의 가로세로 크기를 직접 조절합니다.
  • 드래그해서 아트보드 위치를 옮길 수 있습니다.
  • 여러 개가 있을 때는 전체 재정렬로 화면 순서를 정리합니다.
아트보드 옵션

Enter로 옵션 창 열기

  • 아트보드 툴이 선택된 상태에서 Enter를 누르면 옵션을 열 수 있습니다.
  • 정확한 수치 입력이 필요할 때는 드래그보다 옵션 창이 더 안정적입니다.
아트보드 패널

순서 바꾸기는 패널에서

  • 아트보드 순서 변경은 패널에서만 할 수 있습니다.
  • 패널 순서와 화면 배치 순서가 다를 수 있습니다.
  • 이때 아트보드 재정렬을 하면 패널 순서와 화면 순서를 맞추기 쉽습니다.
수업 포인트 “패널의 순서”와 “눈에 보이는 화면 위치”는 같은 뜻이 아닙니다. 시험이나 과제에서 여러 아트보드를 다룰 때는 순서와 위치를 따로 확인해야 합니다.

선택 툴로 전체를 다루는 방식

Selection Tool은 오브젝트 전체를 한 덩어리로 보는 도구입니다. 이동, 확대·축소, 복제, 그룹 관리처럼 “전체 단위” 작업에 먼저 떠올리면 됩니다.

전체 선택 기준
Shift

가장 많이 섞어 쓰는 보조키

  • 정비율 유지: 비율을 무너뜨리지 않고 크기 조절합니다.
  • 다중 선택: 여러 오브젝트를 하나씩 추가 선택합니다.
  • 선택 해제: 이미 선택된 항목을 Shift로 다시 눌러 제외할 수 있습니다.
비율 유지 다중 선택 선택 해제
Alt

가운데 기준으로 크기 조절

  • Alt를 누른 채 크기를 조절하면 중심을 기준으로 커지거나 작아집니다.
  • Shift와 같이 쓰면 정비율 + 중심 기준 조절을 동시에 만들 수 있습니다.
반복

Ctrl + D로 이전 작업 반복

  • 직전에 했던 이동, 복제, 변형 같은 작업을 다시 반복합니다.
  • 등간격 배치나 패턴 느낌의 반복 작업에서 자주 씁니다.
그룹

묶고 풀기

  • Ctrl + G: 그룹
  • Ctrl + Shift + G: 그룹 해제
  • 여러 개를 한 번에 움직이고 싶을 때는 그룹부터 생각하면 속도가 빨라집니다.
수정 모드

더블클릭으로 안쪽만 편집

  • 오브젝트를 더블클릭하면 수정 모드로 들어갑니다.
  • 그룹 안에 있는 일부만 건드릴 때 유용합니다.
  • 바깥쪽 여백을 더블클릭하면 수정 모드가 해제됩니다.
잠금

실수 방지용 잠금

  • Ctrl + 2: 선택한 오브젝트 잠금
  • Ctrl + Alt + 2: 전체 잠금 해제
  • 배경이나 기준 틀을 고정해두면 선택 실수가 줄어듭니다.
Ctrl + Z

실행 취소. 뒤로 가기처럼 생각하면 됩니다.

Ctrl + Shift + Z

재실행. 되돌린 작업을 다시 앞으로 가져옵니다.

클릭 / 드래그

클릭은 한 개 선택, 드래그는 범위를 묶어서 선택하는 기본 행동입니다.

직접 선택 툴은 점과 선을 만지는 도구

Direct Selection Tool은 전체 오브젝트가 아니라 앵커포인트, 핸들, 패스 일부를 직접 건드리는 툴입니다. 모양 수정 단계에서 쓰며, Selection Tool과 역할이 완전히 다릅니다.

세밀한 수정
핵심 차이

Selection과 Direct Selection 비교

  • Selection: 오브젝트 전체를 이동하고 크기 조절합니다.
  • Direct Selection: 특정 점만 골라 위치를 바꾸거나, 핸들을 꺾고, 선 일부를 수정합니다.
  • 전체가 같이 움직이면 Selection, 일부만 움직이면 Direct Selection이라고 구분하면 됩니다.
전체 단위 점·선 단위
실전 감각

언제 직접 선택으로 넘어가나

  • 모양은 거의 맞았는데 한쪽 선만 더 휘게 하고 싶을 때
  • 앵커포인트 위치를 조금만 옮기고 싶을 때
  • 핸들 방향을 바꿔 곡선을 정리하고 싶을 때

Selection 느낌

점이 아니라 전체 테두리 상자가 보입니다. 크기 조절, 이동, 복제 같은 전체 작업에 맞습니다.

Direct Selection 느낌

앵커포인트와 핸들이 보입니다. 여기서부터는 모양을 직접 다듬는 단계입니다.

앵커포인트, 핸들, 패스 구조

직접 선택을 배우면 반드시 같이 이해해야 하는 개념입니다. 이름만 외우는 게 아니라, 무엇이 어디를 조절하는지 연결해서 기억해야 합니다.

패스 읽기
용어 정리

세 가지를 분리해서 기억

  • 앵커포인트: 패스가 꺾이거나 시작·종료되는 기준점
  • 핸들: 곡선 방향과 휘어짐을 조절하는 막대
  • 패스: 점과 점을 연결하는 선
패스 종류

열린 패스와 닫힌 패스

  • 열린 패스: 시작점과 종료점이 다릅니다.
  • 닫힌 패스: 시작점과 종료점이 같습니다.
  • 채우기 여부나 면이 생기는 구조를 이해할 때 아주 중요합니다.
한 문장으로 정리 “점은 앵커포인트, 점을 잡아 끄는 막대는 핸들, 그 사이를 잇는 선은 패스”로 입에 붙을 때까지 반복해서 말해보면 기억이 빨라집니다.

작업 순서로 다시 외우기

메뉴 이름보다 실제 손 순서로 기억하면 오래 갑니다. 아래 흐름으로 복습하면 시험 문제나 실기에서 바로 떠올리기 좋습니다.

작업 흐름
1단계

아트보드 확인

문서 시작 후 작업 화면 크기와 위치부터 정리합니다. 여러 장이면 패널 순서도 같이 확인합니다.

2단계

Selection으로 전체 배치

오브젝트를 먼저 전체 단위로 잡고 이동, 그룹, 비율 조절, 잠금 같은 기본 배치를 정리합니다.

3단계

Direct Selection으로 미세 수정

전체 위치가 맞으면 점과 핸들을 조절해 곡선과 모양을 다듬습니다.
