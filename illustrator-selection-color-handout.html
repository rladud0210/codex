선택 도구와 색상 핵심 정리

이번 수업에서는 선택 관련 기능과 색상 다루기를 함께 익혔습니다. 실제 작업에서 가장 자주 쓰는 흐름만 모아, 바로 복습할 수 있도록 정리한 자료입니다.

이번 시간 핵심

선택

Move, Lasso, Magic Wand, Group Selection, Same 기능으로 필요한 오브젝트를 빠르게 찾고 선택합니다.

단축키

전체 선택, 선택 해제, 다시 선택, 아트보드 단위 선택을 상황에 맞게 구분해 사용합니다.

레이어 순서

오브젝트의 앞뒤 순서를 단축키로 바꾸되, 이동 범위가 같은 레이어 안이라는 점을 꼭 기억합니다.

색상

스포이드와 Fill / Stroke 순서를 정확히 확인해 원하는 속성만 가져오도록 연습합니다.

빠른 복습 키워드

선택도구 Lasso Tool Magic Wand Tool Group Selection Ctrl + Y Select > Same 레이어 순서 Eyedropper Fill / Stroke

선택 기타 등등

선택 기능은 오브젝트를 정확하게 집는 능력과 연결됩니다. 도구마다 쓰는 상황이 다르니, 무엇을 기준으로 선택하는지 구분해서 익히는 것이 중요합니다.

선택도구

기본 선택 도구로 오브젝트 전체를 선택합니다. 선택 후 Enter를 누르면 이동 관련 설정 창으로 연결해 작업 흐름을 빠르게 이어갈 수 있습니다.

Lasso Tool

정해진 사각형 범위가 아니라, 손으로 그리듯 자유롭게 필요한 부분을 감싸 선택할 때 유용합니다.

Magic Wand Tool

비슷한 속성을 가진 오브젝트를 한 번에 선택할 때 사용합니다. 같은 색상이나 비슷한 속성의 요소를 묶어 잡기 좋습니다.

Group Selection

그룹 안에 들어 있는 오브젝트를 단계적으로 선택할 수 있습니다. 그룹 구조를 해제하지 않고 내부 요소를 집을 때 편리합니다.

Ctrl + Y 윤곽선 모드

색, 효과, 채움보다 경로와 구조를 우선 확인하는 모드입니다. 겹친 오브젝트나 숨은 경로를 파악할 때 특히 유용합니다.

Select > Same

선택한 오브젝트와 같은 속성을 가진 대상을 자동으로 찾아줍니다. 반복 수정 작업에서 시간을 크게 줄여줍니다.

선택 단축키

전체 선택과 선택 해제, 다시 선택은 작업 속도를 좌우하는 기본 동작입니다. 아트보드 전체 선택 단축키와 일반 전체 선택 단축키를 구분해서 사용하세요.

오브젝트가 앞으로 오느냐 뒤로 가느냐에 따라 화면 결과가 크게 달라집니다. 단축키를 외워두면 배치 순서를 빠르게 정리할 수 있습니다.

중요 단축키로 앞뒤 순서를 바꾸는 것은 해당 레이어 안에서만 가능합니다. 레이어 사이를 넘나드는 이동은 되지 않으므로, 다른 레이어로 옮겨야 할 때는 레이어 패널을 열어서 직접 이동해야 합니다.

색상

같은 색을 맞추는 작업은 단순해 보이지만, 실제로는 어떤 속성까지 함께 복사되는지 이해해야 실수를 줄일 수 있습니다.

스포이드

기본적으로 스포이드는 색만 가져오는 것이 아니라 여러 속성을 함께 뽑아옵니다. 그래서 예상보다 많은 스타일이 한 번에 바뀔 수 있습니다.

색만 뽑고 싶다면

Shift를 함께 사용하면 색상만 가져오는 방식으로 작업할 수 있습니다. 불필요한 속성 복사를 줄이는 데 중요합니다.

Fill Color

도형 내부를 채우는 색입니다. 현재 무엇이 앞에 선택되어 있는지 먼저 확인하고 적용해야 합니다.

Stroke Color

선 색상입니다. Fill과 Stroke의 선택 순서를 헷갈리면 원하는 대상이 아닌 다른 속성이 바뀔 수 있습니다.

체크 포인트 1

스포이드 사용 전, 현재 Fill이 선택되어 있는지 Stroke가 선택되어 있는지 먼저 확인합니다.

체크 포인트 2

스타일 전체를 복사할지, 색만 가져올지 작업 목적을 먼저 정하고 스포이드를 사용합니다.

마무리 정리

선택 도구는 정확한 대상 찾기, 레이어 순서는 화면 배치 정리, 색상 기능은 시각 통일과 연결됩니다. 세 기능은 각각 따로가 아니라 한 작업 안에서 계속 함께 쓰입니다.

오브젝트를 찾을 때는 선택 도구와 윤곽선 모드를 함께 활용합니다.
같은 속성을 한 번에 다룰 때는 Magic Wand Tool 또는 Select > Same을 활용합니다.
색상 적용 전에는 Fill / Stroke의 현재 선택 상태를 반드시 확인합니다.
