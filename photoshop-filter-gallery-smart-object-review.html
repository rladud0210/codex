핵심 개념 두 가지

이번 복습의 중심은 “필터 갤러리가 무엇인지”와 “왜 먼저 스마트 오브젝트로 바꾸는지”입니다.

개념 01

필터 갤러리

여러 필터를 한 화면에서 미리 보고 선택하거나 겹쳐 적용할 수 있는 필터 전용 작업 창입니다.

  • 대표적으로 아티스틱, 브러시 스트로크, 스케치 계열 효과를 확인하기 좋습니다.
  • 미리보기 화면을 보면서 수치를 조절할 수 있어 결과를 예측하기 쉽습니다.
  • 효과를 쌓듯이 적용해 스타일을 만들 수 있습니다.
개념 02

보정 전 스마트 오브젝트 처리

필터나 변형을 적용하기 전에 레이어를 스마트 오브젝트로 바꾸면 원본을 유지하면서 다시 수정할 수 있습니다.

  • 원본 픽셀을 직접 망가뜨리지 않는 비파괴 편집이 가능합니다.
  • 필터가 스마트 필터로 적용되어 나중에 옵션을 다시 열 수 있습니다.
  • 마스크와 함께 쓰면 특정 영역만 자연스럽게 조절할 수 있습니다.

실무 순서로 정리

수업 실습이나 시험 작업에서는 순서를 안정적으로 기억하는 것이 중요합니다.

1

보정할 레이어 선택

사진 한 장인지, 여러 레이어를 함께 처리할 것인지 먼저 판단합니다. 필요한 레이어를 선택한 뒤 작업을 시작합니다.

여러 레이어를 한 번에 필터 처리해야 한다면 먼저 병합 복사본이나 그룹 구조를 점검합니다.
2

스마트 오브젝트로 변환

레이어 패널 우클릭 → 스마트 오브젝트로 변환 순서로 처리합니다. 이 단계가 원본 보호의 핵심입니다.

이 과정을 먼저 해두면 이후 필터를 잘못 걸어도 삭제하거나 다시 수정할 수 있습니다.
3

필터 갤러리 열기

필터 → 필터 갤러리로 들어가서 원하는 계열의 효과를 탐색합니다. 미리보기로 이미지 변화를 확인하면서 수치를 조절합니다.

필터가 강하면 전체 톤이 과해질 수 있으므로 한 번에 세게 주기보다 점진적으로 확인하는 편이 안정적입니다.
4

스마트 필터 상태로 다시 점검

적용 후 레이어 아래에 스마트 필터가 생기면 성공입니다. 필요하면 필터 이름을 더블클릭해 옵션을 다시 열고 수정합니다.

효과가 일부만 필요하면 스마트 필터 마스크를 이용해 브러시로 가리고 보이게 조절할 수 있습니다.

비교로 기억하기

시험에서 자주 헷갈리는 지점을 짧게 비교해 두면 실수 방지에 도움이 됩니다.

직접 적용

일반 레이어에 필터 적용

적용은 빠르지만 원본 픽셀이 직접 바뀌기 쉽습니다. 나중에 수치 수정이 제한적입니다.

권장 방식

스마트 오브젝트 후 필터 적용

원본 보존, 재편집 가능, 스마트 필터 마스크 사용 가능이라는 장점이 있습니다.

기억 포인트

필터 갤러리는 미리보기 중심

효과를 탐색하고 조절하는 공간으로 이해하면 됩니다. 적용 전후를 비교하며 판단하는 도구입니다.

자주 나오는 실수

  • 스마트 오브젝트로 바꾸지 않고 바로 필터를 걸어 원본을 되돌리기 어렵게 만드는 경우
  • 필터 강도를 너무 높게 주어 이미지 질감이 부자연스러워지는 경우
  • 필터 갤러리와 다른 필터 메뉴를 구분하지 못해 원하는 효과를 찾지 못하는 경우
  • 스마트 필터가 생겼는데도 다시 수정 가능하다는 점을 활용하지 못하는 경우

작업 전 체크리스트

지금 선택한 레이어가 맞는지 확인했다.
필터 적용 전에 스마트 오브젝트로 변환했다.
필터 갤러리에서 미리보기로 강도를 먼저 확인했다.
적용 후 스마트 필터로 다시 수정 가능한 상태인지 확인했다.
