Core Concepts

벡터와 비트맵

일러스트레이터를 배우기 전에 먼저 이해해야 하는 가장 기본적인 이미지 표현 방식입니다.

# 벡터

수학적 공식으로 만들어지는 이미지

  • 선, 도형, 패스를 계산된 수치로 표현합니다.
  • 확대하거나 축소해도 화질이 깨지지 않습니다.
  • 로고, 아이콘, 캐릭터 라인 작업, 편집 가능한 도형 작업에 유리합니다.
  • 대표 프로그램은 일러스트레이터입니다.
# 비트맵

픽셀로 만들어지는 이미지

  • 작은 색 점인 픽셀이 모여 이미지를 구성합니다.
  • 다채로운 색 표현과 사진 같은 묘사에 강합니다.
  • 확대하면 픽셀이 보이면서 화질이 깨질 수 있습니다.
  • 대표 프로그램은 포토샵입니다.

Color Mode

RGB와 CMYK

파일을 만들 때 색상 모드를 잘못 선택하면 웹에서는 괜찮아 보이던 디자인이 인쇄에서 탁하게 나올 수 있습니다.

RGB 디지털

디지털 화면에서 보는 작업

  • 모니터, 휴대폰, 태블릿처럼 빛으로 보는 화면용 색상 모드입니다.
  • 다채롭고 선명한 색 표현이 가능합니다.
  • 웹배너, 카드뉴스, 상세페이지, SNS 이미지 작업에 적합합니다.
CMYK 인쇄

출력과 제작을 위한 작업

  • 인쇄기 기준의 색상 모드라 RGB보다 색이 다소 칙칙하게 보일 수 있습니다.
  • 실제 출력 결과를 고려한 준비 단계라고 이해하면 됩니다.
  • 포스터, 컵홀더, 패키지, 명함 등 인쇄를 해야 하는 모든 작업에 사용합니다.

수강생 복습 문제

빈칸을 직접 채우면서 오늘 배운 내용을 다시 떠올릴 수 있도록 간단한 복습 문항으로 구성했습니다.

객관식 대신 빈칸 복습

1. 벡터 이미지는 으로 표현된다.

2. 비트맵 이미지는 작은 점인 로 구성된다.

3. 웹배너, 카드뉴스, 상세페이지 작업에는 보통 를 사용한다.

4. 포스터, 컵홀더, 패키지, 명함처럼 인쇄가 필요한 작업은 가 기본이다.

5. 도구들이 모여 있는 인터페이스 영역은 라고 부른다.

점수: 아직 채점 전

짧은 서술형 정리

1. 벡터와 비트맵 중 로고 작업에는 어떤 방식이 더 적합한지, 그 이유를 간단히 적어 보세요.

예시 답안: 로고는 크기가 다양하게 바뀌어도 선명해야 하므로 확대해도 깨지지 않는 벡터 방식이 더 적합하다.

2. RGB와 CMYK를 잘못 선택했을 때 어떤 문제가 생길 수 있는지 한 줄로 적어 보세요.

예시 답안: 웹용 RGB 파일을 인쇄하면 생각보다 색이 탁해질 수 있고, 작업 목적과 결과물이 어긋날 수 있다.
Summary

오늘 내용 한 장 정리

수업이 끝난 뒤 가장 먼저 기억나야 하는 핵심만 다시 묶었습니다.

이것만 기억하면 됩니다

  • 벡터는 수학적 공식, 비트맵은 픽셀이다.
  • 벡터는 화질이 깨지지 않고, 비트맵은 색 표현이 풍부하다.
  • 일러스트레이터는 벡터 기반, 포토샵은 비트맵 기반이다.

작업 전에 먼저 확인할 것

  • 디지털용이면 RGB, 인쇄용이면 CMYK를 먼저 확인한다.
  • Ctrl + N 새 파일, Ctrl + O 파일 열기를 익힌다.
  • 메뉴, 컨트롤, 툴바, 패널, 아트보드의 위치를 구분한다.
