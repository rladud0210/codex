레이어 마스크와 블렌딩 모드

이미지의 일부를 지우거나 살리고, 배경을 자연스럽게 합성하고, 브러시에 밝기 정보를 반영하는 수업 핵심을 한 번에 정리한 복습자료입니다. 원본을 직접 깎지 않는 편집 흐름을 기준으로 기억하세요.

레이어 마스크 블렌딩 모드 이미지 브러시 등록 복습 문제
Layer Mask / Blend Mode
검은색: 지워짐 회색: 반투명 하얀색: 복구 / 유지

핵심 문장: 레이어 마스크는 원본 훼손 없이 보이게 할 부분과 숨길 부분을 제어한다.

1. 레이어 마스크

지우개처럼 보이지만 실제 픽셀을 지우는 것이 아니라, 보이는 정도를 마스크 값으로 기록하는 기능입니다.

무엇을 하는 기능인가?

이미지의 일부분을 숨기거나 다시 드러낼 수 있습니다. 작업 중 수정이 생겨도 다시 칠해서 복구할 수 있습니다.

원본 훼손 없음 수정 가능 부분 편집 가능

명도 인식 규칙

검은색: 지워짐 / 숨겨짐
회색: 반투명 / 부분적으로 보임
하얀색: 복구됨 / 그대로 유지됨

기억해야 할 관점

마스크는 “픽셀 삭제”가 아니라 “보이는 범위 제어”입니다. 검정 브러시로 가리고, 흰 브러시로 다시 살립니다.

  • 검정으로 칠하면 숨김
  • 흰색으로 칠하면 복구
  • 회색은 투명도 조절
STEP 1

선택 영역 만들기

빠른 선택 도구, 펜 도구, 올가미 도구 등으로 남길 부분 또는 제거할 부분의 선택 영역을 먼저 잡습니다.

STEP 2

레이어 마스크 추가

선택 영역이 잡힌 상태에서 레이어 마스크를 적용하면 선택 상태를 기준으로 보이는 영역과 숨겨지는 영역이 나뉩니다.

STEP 3

배경제거로 연결

선택 영역 외의 부분이 숨겨지므로 배경 제거 작업에 바로 활용할 수 있습니다.

수업 포인트 선택 영역이 잡혀 있는 상태에서 레이어 마스크를 만들면, 선택 영역 외의 부분이 숨겨져 배경제거 효과를 얻을 수 있습니다.
실무 감각 지우개보다 마스크를 먼저 쓰면 수정 요청이 들어왔을 때 다시 선택하거나 되돌리기 훨씬 쉽습니다.

2. 블렌딩 모드

레이어가 아래 레이어와 섞이는 방식을 바꿔 주는 기능입니다. 밝고 어두운 정보에 따라 투명하게 보이는 기준이 달라집니다.

Multiply

하얀색에 가까울수록 투명해지고, 어두운 색은 더 남습니다. 흰 배경 제거 느낌으로 자주 사용합니다.

밝을수록 사라짐
  • 밝은 영역은 약해짐
  • 어두운 선이나 질감은 살아남음
  • 스캔 선화, 먹선 합성에 유리
Screen

검은색에 가까울수록 투명해지고, 밝은 빛은 살아납니다. 어두운 배경 위의 빛 효과를 살릴 때 자주 씁니다.

어두울수록 사라짐
  • 검정 배경 제거에 유리
  • 빛, 불꽃, 반짝임 합성에 적합
  • 전체적으로 밝아지는 경향
Overlay

회색을 기준으로 밝은 것은 더 밝게, 어두운 것은 더 어둡게 만듭니다. 대비를 올리는 효과가 큽니다.

중간 회색이 기준
  • 회색 자체는 큰 변화가 적음
  • 밝은 부분은 더 밝아짐
  • 어두운 부분은 더 어두워짐

언제 어떤 모드를 떠올릴까?

  • Multiply: 흰 배경이 있는 드로잉, 질감, 잉크를 합성할 때
  • Screen: 검은 배경의 빛 효과, 렌즈 플레어, 불꽃을 합성할 때
  • Overlay: 명암 대비를 빠르게 강조하고 싶을 때

헷갈리기 쉬운 한 줄 정리

  • Multiply는 밝은 쪽이 빠진다.
  • Screen은 어두운 쪽이 빠진다.
  • Overlay는 회색 기준으로 위아래 대비를 벌린다.

3. 이미지 브러시 등록

브러시로 등록할 이미지는 색상보다 밝기 정보가 중요합니다. 명도에 따라 브러시의 진하기와 투명도가 달라집니다.

브러시 등록 명도 규칙

  • 하얀색: 거의 찍히지 않음
  • 회색: 반투명하게 찍힘
  • 검은색: 가장 진하게 찍힘
해석 방법 브러시 원본 이미지가 검을수록 힘 있게 남고, 밝을수록 약하게 남습니다.
실전 팁 종이 질감이나 번짐 브러시를 만들 때는 검정과 회색의 비율로 농도를 설계하면 됩니다.

4. 시험 직전 암기 포인트

말로 바로 설명할 수 있을 정도로 짧게 정리해 두면 실습 중 판단이 빨라집니다.

검정 / 회색 / 흰색 숨김 / 반투명 / 복구 또는 유지.
선택 후 마스크 선택 영역 외의 부분이 숨겨져 배경제거에 유리하다.
