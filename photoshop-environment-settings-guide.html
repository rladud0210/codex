환경설정과 파일 관리를 한 번에 보는 수업용 가이드

수업 초반에 가장 많이 확인하는 환경설정 열기, 확대·축소, 화면 이동, 이미지 불러오기, 이미지 크기와 캔버스 조절, 저장 방식, 레이어 선택과 그룹 단축키를 빠르게 찾아볼 수 있도록 정리한 배포용 자료입니다.

핵심 요약 바로가기
환경설정 열기

Ctrl + K 로 바로 열어 수업 시작 전 기본 세팅을 맞춥니다.

화면 조작

Alt + 마우스 휠로 확대·축소, Space + 드래그로 화면을 이동합니다.

파일 관리

불러오기, 이미지 크기 변경, 캔버스 조절, 저장과 내보내기 흐름을 구분해서 기억합니다.

레이어 선택

Shift 는 연속 선택, Ctrl 은 개별 선택, Ctrl + G 는 그룹입니다.

환경설정 Ctrl + K 로 시작 화면 조작 확대·축소, 화면 이동, 실행 취소 파일 관리 불러오기, 크기 조절, 저장 레이어 선택 범위, 그룹, 그룹 해제
Preferences

환경설정 열기

수업 중 환경설정 안내가 나오면 가장 먼저 이 단축키부터 실행합니다.

기본 단축키

Ctrl + K

포토샵 환경설정 창을 여는 가장 빠른 방법입니다.

수업에서 기억할 포인트

  • 환경설정은 작업을 시작하기 전에 한 번 확인하면 이후 실습이 훨씬 안정적입니다.
  • 강사가 설정 변경을 안내하면 메뉴를 찾지 말고 먼저 Ctrl + K 를 누릅니다.
View Control

확대·축소와 화면 이동

실습 중 가장 자주 반복하는 조작입니다. 손에 익을수록 작업 속도가 크게 빨라집니다.

기능 방법 설명
확대·축소 Alt + 마우스 휠 화면을 빠르게 확대하거나 축소할 때 사용합니다.
화면 이동 Space + 마우스 왼쪽 클릭 드래그 캔버스 안을 이동하며 다른 위치를 확인할 때 사용합니다.
실행 취소 Ctrl + Z 방금 한 작업을 뒤로 되돌립니다.
재실행 Ctrl + Shift + Z 실행 취소한 작업을 다시 앞으로 진행합니다.
확대·축소와 화면 이동은 도구를 바꾸지 않고도 계속 쓰게 되는 기본 동작이라 초반에 반드시 익혀 두는 것이 좋습니다.
File Management

파일 관리와 크기 조절

이미지를 가져오는 방법부터 이미지 크기, 캔버스, 저장 방식까지 흐름으로 기억하면 실수가 줄어듭니다.

이미지 가져오기

  1. Ctrl + O 로 파일 열기
  2. 파일을 포토샵 탭 쪽으로 드래그해서 열기
  3. 이미지를 캔버스로 직접 드래그해서 가져오기

이미지 크기와 캔버스 크기

  • Image → Image Size: 이미지와 캔버스를 함께 조절합니다.
  • Image → Canvas Size: 캔버스만 조절합니다.
  • 크롭 툴: 보이는 작업 영역을 잘라내며 정리할 때 사용합니다.

저장 메뉴 구분

  • File → Save: 현재 작업 파일을 그대로 저장
  • File → Save As: 다른 이름이나 형식으로 새로 저장
  • File → Save a Copy: 복사본 저장

내보내기

  • File → Export → Save for Web: 웹 업로드용 결과물을 만들 때 자주 사용합니다.
  • File → Save a Copy: 제출용 파일을 따로 분리할 때 유용합니다.
이미지 전체 크기를 바꾸려는지, 작업판만 넓히거나 줄이려는지 먼저 판단한 뒤 Image Size 와 Canvas Size 를 구분해서 써야 합니다.
Artboard

아트보드 기억할 점

메모가 많지 않은 파트라 수업용 핵심만 짧게 정리했습니다.

언제 쓰는가

  • 한 파일 안에서 여러 화면 시안이나 여러 버전을 함께 관리할 때 사용합니다.
  • SNS 카드뉴스, 배너 시안, 화면별 디자인 정리에 특히 편리합니다.

수업 포인트

  • 아트보드는 여러 작업판을 나누는 개념이고, 레이어는 요소를 쌓아 관리하는 개념입니다.
  • 아트보드와 레이어를 혼동하지 않도록 역할을 분리해서 기억합니다.
Layers

레이어 선택과 그룹

레이어를 정확하게 선택하고 묶을 수 있어야 수정 속도와 정리 상태가 좋아집니다.

기능 단축키 기억할 점
연속 범위 선택 Shift 이 레이어부터 저 레이어까지 이어서 선택할 때 사용합니다.
개별 선택 추가 Ctrl 떨어져 있는 레이어를 하나씩 골라 선택할 때 사용합니다.
그룹 Ctrl + G 선택한 레이어를 한 묶음으로 정리합니다.
그룹 해제 Ctrl + Shift + G 묶어 둔 그룹을 다시 풀 때 사용합니다.
레이어가 많아질수록 그룹 정리는 선택이 아니라 필수입니다. 수정 범위를 줄이고 파일을 읽기 쉽게 만드는 기본 습관으로 보시면 됩니다.
Worksheet

수강생 복습 문제

수업 직후 바로 풀어보면서 단축키와 메뉴 흐름이 기억나는지 점검하는 파트입니다.

빈칸 채우기

1. 포토샵 환경설정을 여는 단축키를 쓰세요.

2. 화면 확대·축소 방법을 쓰세요.

3. 화면 이동 방법을 쓰세요.

4. 이미지와 캔버스를 함께 조절하는 메뉴명을 쓰세요.

5. 캔버스만 조절하는 메뉴명을 쓰세요.

6. 선택한 레이어를 그룹으로 묶는 단축키를 쓰세요.

점수: 아직 채점 전

서술형 확인

1. Save, Save As, Save a Copy 의 차이를 간단히 적어보세요.

예시 답안: Save는 현재 파일에 그대로 저장, Save As는 다른 이름이나 형식으로 새 저장, Save a Copy는 원본과 별도로 복사본 저장이다.

2. 아트보드와 레이어의 차이를 한 줄로 정리해보세요.

예시 답안: 아트보드는 여러 작업판을 나누는 개념이고, 레이어는 한 작업판 안에서 요소를 층층이 관리하는 개념이다.

3. Shift 선택과 Ctrl 선택이 각각 어떤 상황에서 쓰이는지 적어보세요.

예시 답안: Shift는 연속된 레이어 범위를 한 번에 선택할 때, Ctrl은 떨어져 있는 레이어를 개별로 추가 선택할 때 사용한다.
