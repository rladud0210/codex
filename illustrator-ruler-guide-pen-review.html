단축키 먼저 정리

눈금자는 기준을 만들기 위한 시작점이고, 가이드는 실제 배치 기준선입니다. 만든 가이드는 보이기와 숨기기를 자주 오가며 작업합니다.

보기 · 가이드
Ctrl + R 눈금자 켜기 / 끄기 상단과 왼쪽 기준자를 보여 주거나 숨깁니다.
Ctrl + ; 가이드 켜기 / 끄기 화면의 가이드 표시만 숨기거나 다시 보여 줍니다.
Ctrl + 5 가이드 만들기 선택한 선이나 도형을 가이드로 바꿉니다.

모양 만들기 전 체크리스트

완성된 모양만 보고 바로 따라 그리기보다, 어떤 기본 도형에서 시작했는지 역으로 분석합니다.

분석 순서

기본 형태 확인

모양이 편집되기 전 기본 형태가 무엇이었는지 먼저 봅니다.

필요 기능 선택

저 모양을 만들려면 패스파인더, 도형 구성 도구, 펜툴 중 무엇을 써야 하는지 판단합니다.

선택 부분 결정

어떤 부분을 선택해야 원하는 부분이 남고, 어떤 부분이 잘릴지 확인합니다.

펜툴 복습

펜툴은 예쁘게 끌어당기는 도구가 아니라, 정확한 위치에 앵커포인트를 찍고 방향선을 통제하는 도구입니다.

앵커포인트 · 핸들

정확한 위치

앵커포인트를 곡률이 시작하는 부분과 끝나는 부분에 찍습니다.

직선으로 먼저 보기

처음부터 곡선이라고 생각하지 말고, 큰 직선 흐름으로 구조를 먼저 봅니다.

핸들 유지

곡선을 그리는 와중에 특별한 꺾임이 없다면 핸들을 끊지 않습니다.

최소 개수

앵커포인트 개수는 최소한으로 사용하고, 방향선은 항상 진행 방향으로 뻗게 둡니다.
