Adobe Photoshop Selection Handout

선택 도구와 선택 영역 정리

Object Selection, Quick Selection, Magic Wand, Lasso의 차이와 선택 영역을 수정하는 방법, 자주 쓰는 단축키를 한 장으로 정리한 실습 노트입니다.

색 기준 선택 Magic Wand, Color Range처럼 색상 차이를 기준으로 영역을 잡습니다.
개체 기준 선택 Object Selection처럼 사람, 물건, 배경 요소를 자동 인식해서 잡습니다.
직접 보정 Quick Selection, Lasso, Select and Mask로 부족한 경계와 털을 정리합니다.

1. 선택 도구 4가지

Object Selection Tool
  • 개체를 자동으로 인식해서 선택합니다.
  • 사람, 상품, 물건처럼 형태가 분명한 대상을 빠르게 잡을 때 좋습니다.
  • 처음 큰 영역을 잡고 Quick Selection이나 Lasso로 보정하면 효율적입니다.
Quick Selection Tool
  • 색과 질감을 문지르듯 드래그해서 선택합니다.
  • 브러쉬 크기는 [, ]로 조절합니다.
  • 부족한 부분은 Shift로 추가하고, 넘친 부분은 Alt로 뺍니다.
Magic Wand Tool
  • 클릭한 색상과 비슷한 영역을 선택합니다.
  • 단색 배경이나 색 구분이 뚜렷한 이미지에 적합합니다.
  • 색 차이가 복잡한 사진에서는 선택이 거칠 수 있습니다.
Lasso Tool
  • 직접 그려서 자유롭게 선택 영역을 만듭니다.
  • 자동 선택이 잘 안 되는 작은 부분을 직접 보정할 때 사용합니다.
  • 정교한 외곽선보다는 빠른 수정과 임시 선택에 알맞습니다.

2. Shift와 Alt의 역할

Shift

  • 선택 도구 사용 중: 선택 영역 추가, 다중 선택
  • 자유 변형 사용 중: 정비율 유지

Alt

  • 선택 도구 사용 중: 선택 영역 빼기
  • 자유 변형 사용 중: 가운데 기준으로 크기 조절

3. 자주 쓰는 단축키

Ctrl + D

선택 영역 해제

Ctrl + J

선택한 영역을 새 레이어로 복제

Ctrl + T

자유 변형

Enter

변형 적용

Ctrl + Shift + I

선택 영역 반전

[

브러쉬 크기 축소

]

브러쉬 크기 확대

Caps Lock

브러쉬 커서 모양 변경

4. 선택 영역 수정

레이어 썸네일 선택 Ctrl을 누른 채 레이어 썸네일을 클릭하면 그 레이어의 불투명한 부분이 그대로 선택 영역으로 잡힙니다.
Select > Modify > Expand 선택 영역을 바깥쪽으로 확대합니다.
Select > Modify > Contract 선택 영역을 안쪽으로 축소합니다.
Select > Modify > Feather 선택 영역의 가장자리를 부드럽게 흐립니다.
선택 영역 반전 Ctrl + Shift + I로 선택된 영역과 선택되지 않은 영역을 반대로 바꿉니다.

털과 머리카락 정리

  • Select > Select and Mask를 사용합니다.
  • 선택 도구 사용 중 옵션바의 Select and Mask 버튼으로도 들어갈 수 있습니다.
  • 머리카락, 털, 복잡한 가장자리처럼 일반 선택 도구만으로 깔끔하게 잡기 어려운 부분을 보정합니다.

5. Color Range 이해

흰색

선택 영역으로 잡힙니다.

회색

반투명하게 선택됩니다.

검은색

선택되지 않습니다.

6. 실습 순서

  1. Object Selection Tool로 큰 개체를 먼저 선택합니다.
  2. Quick Selection Tool로 부족한 부분을 추가합니다.
  3. Alt를 누른 채 넘친 선택 영역을 제거합니다.
  4. Select and Mask로 털, 머리카락, 복잡한 가장자리를 정리합니다.
  5. Ctrl + J로 선택 영역을 새 레이어로 복제합니다.
  6. Ctrl + T로 크기와 위치를 조정하고 Enter로 적용합니다.
  7. 컬러와 브러쉬로 필요한 부분을 보정합니다.
