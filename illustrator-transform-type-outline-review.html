1. 변형과 왜곡

도형이나 텍스트를 원근감 있게 바꾸는 기본 흐름을 먼저 익힙니다.

핵심 순서

사다리꼴 만들기 흐름

  1. Free Transform을 선택합니다.
  2. 상단 또는 옵션에서 Perspective를 사용합니다.
  3. 모서리를 드래그해서 위아래 폭이 다른 형태로 바꿉니다.
  4. 직사각형이 사다리꼴처럼 보이면 원근감이 적용된 것입니다.

왜곡 포인트

  • 변형은 모양만 바꾸는 것이 아니라 시선 방향을 만드는 작업입니다.
  • Perspective는 평면을 기울여 보이게 할 때 유용합니다.
  • 텍스트에도 적용할 수 있지만, 수정 단계인지 확정 단계인지 먼저 판단해야 합니다.

2. 텍스트 기본 정리

문자 입력 상태와 문자 객체 상태를 구분해야 이후 작업이 꼬이지 않습니다.

Type 메뉴

텍스트 작성 후 확정

  • 텍스트를 입력한 뒤 Ctrl + Enter로 문자 입력을 확정합니다.
  • 확정 전에는 입력 중인 상태이고, 확정 후에는 객체처럼 선택할 수 있습니다.

대문자 변경

  • Type > Change Case 메뉴를 사용합니다.
  • 영문 제목 정리나 표기 통일이 필요할 때 빠르게 처리할 수 있습니다.

텍스트 종류 전환

  • 드래그로 만들면 영역 텍스트, 클릭으로 만들면 포인트 텍스트가 됩니다.
  • Type > Convert to Point Text
  • Type > Convert to Area Text

헷갈리기 쉬운 구분

드래그 → 클릭 / 클릭 → 드래그가 아니라, 이미 만든 텍스트 객체의 형식을 Point Text와 Area Text 사이에서 변환하는 개념입니다.

복습 질문

  • 지금 선택한 텍스트는 입력 중인가?
  • 포인트 텍스트인가, 영역 텍스트인가?
  • 변형 전에 수정 가능한 상태를 유지해야 하는가?

3. 도형화: 윤곽선 만들기

이번 페이지에서 가장 중요한 부분입니다. 별표가 많은 이유가 있습니다.

★★★★★

윤곽선 만들기

  • Type > Create Outlines
  • 단축키: Ctrl + Shift + O
  • 의미: 글자를 더 이상 문자로 보지 않고 도형(Path)으로 바꿉니다.
  • 수업 표현으로는 글자를 '깬다'고 기억하면 됩니다.

왜 중요할까

  • 문자 상태에서는 자유로운 패스 편집이 제한됩니다.
  • 왜곡, 합치기, 나누기, 쉐이프 가공을 하려면 도형 상태가 더 유리합니다.
  • 출력 환경에 폰트가 없어도 글자 모양을 유지하기 좋습니다.

반드시 기억

글자는 글자인 상태로는 패스 편집을 자유롭게 하기 어렵습니다.

도형처럼 다루려면 윤곽선 만들기를 해야 합니다.

Fill 컬러가 채워져 있다고 해서 선은 아닙니다. 채워진 면과 실제 Stroke는 구분해서 봐야 합니다.

교재 메모

  • 264p 내용과 함께 다시 확인
  • 문자 상태와 윤곽선 상태를 비교해서 보기

4. 단축키와 실무 연결

도형화 이후에는 복제, 잠금, 패스 조합이 자주 이어집니다.

빠른 작업

자주 쓰는 단축키

복사 Ctrl + C
잠금 Ctrl + 2
제자리에 붙여넣기 Ctrl + F
잠금 해제 Ctrl + Alt + 2

함께 묶어서 기억할 도구

  • Pathfinder: 여러 도형을 합치거나 빼고 나눌 때 사용
  • Shape Builder: 드래그 기반으로 직관적으로 영역을 합치거나 제거
  • Offset Path: 기존 패스를 기준으로 안쪽 또는 바깥쪽에 새 경로 생성
윤곽선으로 만든 텍스트는 위 도구들과 연결해서 로고형 타이포, 배지, 스티커 형태 작업으로 자주 이어집니다.
