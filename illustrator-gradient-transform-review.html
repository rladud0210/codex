1. 그라디언트 삽입

채우기 색에 그라디언트를 넣고, 그라디언트 툴로 방향과 범위를 조절합니다.

.

그라디언트 삽입

선택한 오브젝트의 채우기 색에 기본 그라디언트를 바로 적용합니다.

G

그라디언트 툴

드래그해서 그라디언트의 시작점, 끝점, 방향, 길이를 직접 조절합니다.

Fill

적용 위치 확인

그라디언트가 면에 들어가는지, 선에 들어가는지 먼저 확인해야 합니다.

핵심: 그라디언트가 이상하게 보이면 먼저 Fill/Stroke 중 어디가 활성화되어 있는지 확인합니다.

2. 색 우선순위 체크

색을 바꾸기 전에 어떤 색이 어디에 적용되어 있는지 확인하면 작업이 덜 꼬입니다.

색 확인 순서

  1. 1오브젝트가 정확히 선택되어 있는지 확인
  2. 2Fill과 Stroke 중 어느 쪽이 앞으로 나와 있는지 확인
  3. 3그라디언트 색상 정지점이 선택되어 있는지 확인
  4. 4전체 색을 바꿀지, 특정 색만 바꿀지 결정

Recolor Artwork 경로

작품 전체의 색을 다시 배치하거나, 선택한 색상을 한 번에 바꿀 때 사용합니다.

Edit Edit Colors Recolor Artwork
색 우선순위를 먼저 체크한 뒤 Recolor Artwork를 열면 원하는 색만 바꾸기 쉽습니다.

3. 변형은 축 기준

회전, 반전, 크기조절은 모두 기준점, 즉 축을 어디에 두는지가 결과를 결정합니다.

축을 기준으로 변형
조작 의미 기억 포인트
클릭 후 드래그 축을 기준으로 자유롭게 이동하며 변형 눈으로 보면서 방향과 정도를 맞출 때 사용
Enter 원위치에서 수치 입력으로 변형 정확한 각도, 비율, 위치가 필요할 때 사용
Alt 클릭 축을 이동한 뒤 그 축을 기준으로 변형 중심이 아닌 다른 지점을 기준으로 돌리거나 복제할 때 중요

회전

축을 중심으로 오브젝트를 돌립니다. 기준점이 바뀌면 회전 궤적도 바뀝니다.

반전

축을 기준으로 좌우 또는 상하 방향을 뒤집습니다.

크기조절

축을 중심으로 커지거나 작아집니다. 선과 모서리 옵션을 함께 확인합니다.

4. Scale 옵션

크기조절할 때 선 두께와 모서리 곡률이 같이 변할지, 그대로 유지될지를 결정합니다.

Scale Stroke & Effects ON

선 두께도 같은 비율로 조절

키면 오브젝트 크기에 맞춰 선 두께와 효과도 같이 커지거나 작아집니다.

Scale Stroke & Effects OFF

선 두께 유지

끄면 오브젝트 크기만 바뀌고 선 두께와 효과 크기는 유지됩니다.

Scale Corners ON

곡률도 같은 비율로 조절

키면 둥근 모서리의 곡률이 크기 변화에 맞춰 같이 변합니다.

Scale Corners OFF

곡률 유지

끄면 크기를 조절해도 모서리 둥글기는 기존 값에 가깝게 유지됩니다.

암기 문장: Scale Stroke & Effects는 선 두께, Scale Corners는 모서리 곡률을 같이 키울지 정하는 옵션입니다.

5. 복습 문제

빈칸에 직접 입력하고 채점해보세요. 정답 보기와 초기화, PDF 출력까지 사용할 수 있습니다.

빈칸 문제

1. 선택한 오브젝트에 그라디언트를 삽입하는 단축키는 무엇인가?

2. 그라디언트 툴을 선택하는 단축키는 무엇인가?

3. 색이 이상하게 적용될 때 먼저 확인해야 하는 두 위치는 무엇인가?

4. Edit → Edit Colors → 다음에 들어가는 색상 재배치 메뉴 이름은?

5. 회전, 반전, 크기조절은 모두 무엇을 기준으로 변형되는가?

6. 축을 이동한 뒤 그 축을 기준으로 변형하려면 어떤 조작을 사용하는가?

7. 크기 조절 시 선 두께와 효과도 같은 비율로 조절하는 옵션은?

8. 크기 조절 시 모서리 곡률도 같은 비율로 조절하는 옵션은?

점수: 아직 채점 전

서술형 문제

1. 그라디언트 색이 원하는 곳에 적용되지 않을 때 확인할 순서를 적어보세요.

예시 답: 먼저 오브젝트가 선택되어 있는지 확인하고, Fill과 Stroke 중 어느 쪽이 활성화되어 있는지 본다. 그다음 그라디언트 정지점이나 Recolor Artwork에서 바꿀 색을 확인한다.

2. Alt 클릭으로 축을 옮겨 변형하는 상황을 한 가지 예로 설명해보세요.

예시 답: 꽃잎이나 시계 눈금처럼 특정 중심점을 기준으로 여러 개를 회전 복제해야 할 때 Alt 클릭으로 기준축을 옮긴 뒤 회전 값을 입력한다.

3. Scale Stroke & Effects를 꺼야 하는 상황과 켜야 하는 상황을 비교해보세요.

예시 답: 선 두께를 일정하게 유지해야 하면 끄고, 오브젝트 전체가 같은 비율로 커진 느낌을 유지해야 하면 켠다.

4. 오늘 내용 중 실제 작업 전에 꼭 확인할 것 3가지를 적어보세요.

예시 답: Fill/Stroke 활성 위치, 색 우선순위, 변형 기준축 위치, Scale Stroke & Effects, Scale Corners.

마무리 체크

작업 전후로 아래 항목만 확인해도 실수를 줄일 수 있습니다.

  • . 키로 그라디언트를 넣을 수 있다.
  • G 키로 그라디언트 방향과 범위를 조절할 수 있다.
  • Fill과 Stroke 중 어떤 색이 활성화되어 있는지 확인한다.
  • Edit → Edit Colors → Recolor Artwork 경로를 기억한다.
  • 변형은 축을 기준으로 일어난다는 점을 기억한다.
  • Alt 클릭은 축 이동과 변형 기준 설정에 중요하다.
  • Scale Stroke & Effects를 켜면 선 두께도 같이 조절된다.
  • Scale Corners를 켜면 모서리 곡률도 같이 조절된다.
오늘의 핵심: 색을 바꾸기 전에는 Fill/Stroke와 색 우선순위를 확인하고, 변형하기 전에는 축 위치와 Scale 옵션을 먼저 확인합니다.
