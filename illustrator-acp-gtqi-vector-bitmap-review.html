1. 자격증 비교

시험 이름만 외우지 말고, 어떤 방식으로 평가하는지까지 함께 기억해야 실제 준비 방향이 잡힙니다.

시험 구조 먼저
ACP

국제공인 자격증

  • 구분: 국제공인
  • 구성: 실기 + 필기 33문항
  • 시간: 50분
  • 난이도 메모: 하
  • 시험 일정 메모: 5/6-5/7, 13:00-14:00
GTQi

1급 · 2급 · 3급

  • 등급: 1급, 2급, 3급
  • 구성: 실기 3문제
  • 시간: 1시간 30분
  • 난이도 메모: 중상
컴그

컴퓨터그래픽스운용기능사

  • 구성: 필기 + 실기
  • 실기: 1문제
  • 시간: 3시간 30분

빠른 기억법: ACP는 짧고 가볍게 구조 이해, GTQi는 정해진 문제를 시간 안에 처리, 컴그는 긴 시간 동안 한 문제를 끝까지 완성하는 흐름으로 구분하면 정리가 빠릅니다.

2. 벡터와 비트맵

이미지 표현 방식을 구분하는 것은 프로그램 선택의 출발점입니다. 일러스트레이터와 포토샵이 왜 다른지 여기서 갈립니다.

이미지 표현 방식

핵심 기준

벡터 = 수학적 공식 비트맵 = 픽셀
벡터

수학적 공식으로 모양을 계산해서 그립니다.

  • 장점: 확대해도 화질이 깨지지 않음
  • 단점: 사진처럼 복잡한 색 표현에는 한계가 있음
  • 대표 프로그램: 일러스트레이터
비트맵

작은 픽셀이 모여 이미지를 만듭니다.

  • 장점: 다채로운 색 표현 가능
  • 단점: 확대하면 화질이 깨질 수 있음
  • 대표 프로그램: 포토샵
수업 포인트

로고, 아이콘, 도형 중심 작업은 벡터가 유리하고, 사진 보정이나 사실적인 이미지 작업은 비트맵이 유리합니다.

바로 연결해서 기억하기

Illustrator 도형, 로고, 아이콘, 편집이 쉬운 선 작업
Photoshop 사진, 질감, 섬세한 색 보정, 합성 작업

3. RGB와 CMYK

어디에 쓰는 결과물인지 먼저 판단하면 색 모드를 고르기 쉬워집니다. 디지털용인지, 인쇄용인지부터 묻는 습관이 중요합니다.

색 모드 구분
RGB

디지털 화면용

빛의 색을 사용하는 방식이라 화면에서 다채로운 색 표현이 가능합니다.

  • 웹배너
  • 카드뉴스
  • 상세페이지
  • 그 외 화면에서 보는 작업물
CMYK

인쇄용

인쇄 과정에서 화면보다 색이 칙칙해질 수 있으므로, 인쇄물을 만들 때는 미리 이 모드로 작업하는 것이 안전합니다.

  • 포스터
  • 컵홀더
  • 패키지
  • 명함 등 인쇄해야 하는 모든 것
구분 RGB CMYK
기준 디지털 화면 인쇄 결과물
색 느낌 더 선명하고 다채롭게 보이기 쉬움 상대적으로 칙칙하게 보일 수 있음
대표 작업 웹배너, 카드뉴스, 상세페이지 포스터, 패키지, 명함, 컵홀더

4. 기본 조작과 인터페이스

짧은 단축키와 저장 방식은 실습 속도를 크게 바꿉니다. 여기서는 가장 자주 먼저 쓰는 명령만 묶어서 정리합니다.

기본기 점검
1

새 파일

Ctrl + N으로 새 파일을 만듭니다. 작업 시작 전 문서 크기와 색 모드를 확인하는 습관까지 함께 가져가야 합니다.

2

열기

Ctrl + O로 기존 파일을 엽니다. 기존 작업 수정, 자료 불러오기, 이어서 작업할 때 가장 기본이 되는 명령입니다.

3

저장

Save, Save As, Save a Copy 차이를 구분해야 합니다. 원본을 덮어쓸지, 다른 이름으로 저장할지 판단하는 단계입니다.

4

이미지 저장

이미지를 따로 내보낼 때는 Export를 사용합니다. 작업 파일 저장과 결과물 내보내기를 혼동하지 않아야 합니다.

인터페이스

일러스트 인터페이스를 볼 때

  • 도구 패널이 어디 있는지
  • 상단 옵션바가 무엇을 바꾸는지
  • 레이어, 속성, 색상 패널 위치
  • 현재 문서의 색 모드와 아트보드 상태
실수 방지

헷갈리기 쉬운 구분

  • Save: 현재 파일 그대로 저장
  • Save As: 다른 이름 또는 다른 위치로 저장
  • Save a Copy: 복사본 저장
  • Export: 이미지 결과물로 내보내기

6. 정답과 마무리

먼저 스스로 풀고 난 뒤에 열어보세요. 정답만 보는 것보다 왜 그 답인지 설명할 수 있어야 복습이 됩니다.

복습 문제 정답
  1. 문제 1: b
  2. 문제 2: 포토샵
  3. 문제 3: RGB
  4. 문제 4: a, c, d
  5. 문제 5: Ctrl + N
  6. 문제 6: Export
오늘의 핵심
  • 벡터는 수학적 공식, 비트맵은 픽셀이다.
  • RGB는 디지털 화면용, CMYK는 인쇄용이다.
  • 작업 파일 저장과 결과물 내보내기는 다르다.
  • 기본 단축키와 저장 구조를 먼저 익혀야 실습 속도가 붙는다.

인쇄 시에는 정답 영역이 함께 표시됩니다.

정리 문장: 일러스트 수업 기초에서는 자격증 종류를 구분하는 것보다, 어떤 작업에 어떤 표현 방식과 색 모드를 써야 하는지 정확히 아는 것이 더 중요합니다. 벡터와 비트맵, RGB와 CMYK, Save와 Export 차이만 정확히 잡아도 실습 실수가 크게 줄어듭니다.
