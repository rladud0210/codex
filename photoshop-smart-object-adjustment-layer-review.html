원본은 지키고, 보정은 나중에도 다시 수정할 수 있게 쌓아가는 방식이 핵심입니다.

1. 스마트 오브젝트를 먼저 이해하기

필터와 보정을 안전하게 다루려면 먼저 비파괴 편집 흐름을 이해해야 합니다.

핵심 개념

스마트 오브젝트란?

원본 픽셀을 직접 망가뜨리지 않고, 변형이나 필터를 나중에도 다시 수정할 수 있게 보존하는 레이어 방식입니다. 확대, 축소, 필터 적용, 재수정이 많은 작업에서 특히 중요합니다.

레이어를 우클릭한 뒤 스마트 오브젝트로 변환으로 시작합니다.
왜 필요한가

필터를 다시 손볼 수 있다

  • 원본 레이어를 직접 훼손하지 않습니다.
  • 필터 적용 뒤에도 수치를 다시 열어서 수정할 수 있습니다.
  • 크기를 줄였다가 다시 키워도 일반 레이어보다 안전합니다.
  • 수업 실습처럼 여러 버전을 비교할 때 훨씬 유리합니다.
1

레이어 선택

보정하거나 필터를 줄 이미지 레이어를 먼저 확인합니다.

2

스마트 오브젝트 변환

우클릭 후 스마트 오브젝트로 변환합니다.

3

필터 또는 보정 적용

필터 갤러리, Camera Raw, 각종 보정을 적용합니다.

4

나중에 다시 수정

레이어 아래 생긴 항목을 더블클릭해서 수치를 다시 바꿉니다.

2. Ctrl + I와 Image > Adjustments

직접 보정과 비파괴 보정의 차이를 같이 기억해야 작업 흐름이 깔끔해집니다.

단축키

Ctrl + I 는 색 반전

현재 선택된 레이어나 마스크의 밝고 어두운 값을 반대로 바꿉니다. 사진 자체를 반전할 수도 있고, 마스크를 반전해 보이는 영역과 가려지는 영역을 뒤집을 때도 자주 씁니다.

이미지에 적용 색이 보색 관계로 뒤집힙니다.
마스크에 적용 검정과 흰색 영역이 뒤바뀝니다.
메뉴 위치

Image > Adjustments

밝기/대비, 색조/채도, 흑백, 색상 균형 같은 보정을 현재 레이어에 직접 적용하는 메뉴입니다. 빠르게 결과를 볼 수 있지만, 일반 레이어에 바로 쓰면 원본 픽셀이 바뀌기 때문에 주의가 필요합니다.

직접 보정이 필요할 때도 가능하면 스마트 오브젝트나 조정 레이어와 함께 쓰는 습관이 안전합니다.
Image > Adjustments 현재 선택한 레이어에 직접 적용됩니다. 일반 레이어에 바로 적용하면 원본 정보가 바뀝니다.
조정 레이어 보정값이 별도 레이어로 생성됩니다. 나중에 수치를 다시 바꾸기 쉽고, 블렌딩 모드와 오퍼시티도 함께 활용할 수 있습니다.
실무 권장 빠른 테스트를 제외하면 조정 레이어 중심으로 작업하는 편이 더 안전하고 수정도 편합니다.

3. 조정 레이어 5가지 핵심

아래 다섯 문장은 수업에서 그대로 외워도 될 정도로 중요합니다.

1

자동으로 레이어가 생성된다

조정값이 이미지에 직접 박히지 않고, 별도 레이어로 쌓입니다.

2

아이콘 더블클릭으로 다시 수정 가능

작업 후에도 언제든지 수치를 열어서 보정 강도를 다시 바꿀 수 있습니다.

3

레이어 기능을 그대로 활용

블렌딩 모드, 오퍼시티, 마스크 등 레이어에 적용되는 기능을 모두 함께 쓸 수 있습니다.

4

아래 레이어들에 영향을 준다

조정 레이어 위치를 기준으로, 그 아래에 있는 레이어들이 함께 보정됩니다.

5

특정 레이어만 보정하려면 클리핑 마스크

조정 범위를 한 레이어로 제한하고 싶을 때는 클리핑 마스크를 사용합니다.

실전 팁

보정 레이어 순서도 중요하다

예를 들어 노출을 먼저 잡고, 그 위에서 색조/채도와 색상 균형을 조절하면 결과를 더 안정적으로 만들 수 있습니다.

4. 클리핑 마스크를 언제 쓰는가

전체 보정과 부분 보정을 구분하지 못하면 레이어가 금방 복잡해집니다.

전체 보정

사진 전체 분위기를 통일할 때

  • 하늘, 인물, 배경을 같이 차갑게 만들고 싶을 때
  • 전체 대비를 높이거나 전체 채도를 낮추고 싶을 때
  • 장면 전체 톤을 한 방향으로 밀고 싶을 때
부분 보정

특정 레이어만 만지고 싶을 때

  • 인물 피부만 밝게 하고 싶을 때
  • 텍스트나 오브젝트 한 개만 색을 바꾸고 싶을 때
  • 합성한 소스 하나만 어둡게 맞추고 싶을 때
조정 레이어 생성 후 클리핑 마스크를 걸면 바로 아래 레이어 한 장에만 적용됩니다.

5. 영화 톤 보정 실습 방향

요청 문장을 받았을 때 어떤 보정을 떠올려야 하는지 연습하는 구간입니다.

실습 A

일본 청춘영화의 청량함

가볍고 맑은 공기감, 부드러운 밝기, 살짝 푸른 하이라이트를 떠올리면 됩니다.

  • 노출 또는 밝기를 약간 올립니다.
  • 채도는 과하지 않게, 대신 밝고 깨끗한 느낌으로 정리합니다.
  • 색상 균형에서 하이라이트를 살짝 시원한 쪽으로 보냅니다.
  • 대비를 너무 세게 주기보다 부드럽게 유지합니다.
  • 흰색 영역이 탁하지 않게 정리하면 분위기가 살아납니다.
실습 B

느와르 영화의 어둡고 강렬한 느낌

명암 대비, 깊은 그림자, 무게감 있는 색을 먼저 생각해야 합니다.

  • 전체 노출을 낮추고, 블랙을 더 깊게 만듭니다.
  • 대비를 높여 명암 차이를 강하게 만듭니다.
  • 채도는 약간 절제하거나 특정 색만 남겨도 좋습니다.
  • 차가운 그림자나 녹청 계열 톤을 섞으면 분위기가 강해집니다.
  • 필요하면 비네트 느낌을 추가해 시선을 모읍니다.
