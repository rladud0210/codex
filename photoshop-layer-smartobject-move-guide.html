Layer Basics

레이어 핵심 정리

레이어는 포토샵 작업의 기본 단위입니다. 각각 따로 움직이고, 숨기고, 수정할 수 있기 때문에 작업을 안전하고 깔끔하게 유지할 수 있습니다.

레이어 선택

원하는 레이어를 정확히 선택해야 이동, 정렬, 변형, 보정이 올바르게 적용됩니다.

레이어 이동

선택한 레이어만 움직이기 때문에 현재 어떤 레이어가 활성화되어 있는지 항상 확인해야 합니다.

레이어 위치

위에 있는 레이어가 아래 레이어를 덮습니다. 순서 변경은 화면 결과에 직접 영향을 줍니다.

레이어 정리 습관

그룹과 이름 정리를 함께 하면 복잡한 작업에서도 수정이 쉬워집니다.

Shortcuts

레이어 단축키 모음

자주 쓰는 명령은 손에 익혀 두면 작업 속도가 크게 달라집니다.

그룹

  • 그룹 만들기: Ctrl + G
  • 그룹 해제: Ctrl + Shift + G

복제

  • 레이어 복제: Ctrl + J

병합

  • 레이어 병합: Ctrl + E
  • 레이어 병합 복제: Ctrl + Alt + Shift + E
  • 보이는 레이어 병합: Ctrl + Shift + E

레이어 순서 이동

  • 한 칸 위로: Ctrl + ]
  • 한 칸 아래로: Ctrl + [
  • 맨 위로: Ctrl + Shift + ]
  • 맨 아래로: Ctrl + Shift + [

개념 확인

1. Auto Select는 왜 평소에는 꺼두고 사용하는 것이 좋은가?
예시 답: 원하지 않는 레이어가 선택되는 실수를 줄이고, 필요한 레이어만 정확하게 선택하기 위해서.
2. 이동도구를 사용할 때 반드시 확인해야 하는 3가지를 써보세요.
예시 답: 선택된 레이어가 맞는지, 이동하는 대상이 맞는지, 위치와 정렬 상태가 의도한 결과와 같은지.
3. 스마트오브젝트가 일반 레이어보다 유리한 이유를 2가지 써보세요.
예시 답: 원본 보호가 쉽고, 크기 조절 시 화질 손상이 적으며, 보정 내용을 다시 수정하기 쉽다.
4. 스마트오브젝트에서 브러시나 지우개 작업이 바로 어려운 이유는 무엇인가?
예시 답: 원본 픽셀을 직접 훼손하지 않도록 보호하는 구조이기 때문이다.
5. 스마트오브젝트가 특히 많이 활용되는 작업을 2가지 써보세요.
예시 답: 보정, 목업, 합성 작업.

서술형 문제

작업하던 이미지의 원본을 최대한 보호하면서 크기 조절과 보정을 반복해야 한다면, 왜 스마트오브젝트를 사용하는 것이 좋은지 설명해 보세요.
예시 답: 원본을 보호한 상태로 크기를 여러 번 조절해도 화질 손상이 적고, 보정 내역을 다시 수정하거나 삭제할 수 있어서 작업이 안전하고 유연하다.

실습 체크 문제

1. 도형 3개를 만든 뒤 그룹으로 묶어보세요.
2. 그룹을 해제한 뒤 레이어 하나를 복제해보세요.
3. 복제한 레이어를 맨 위로 올려보세요.
4. 이미지를 불러와 스마트오브젝트인지 확인해보세요.
5. Ctrl + T로 축소 후 다시 확대해 차이를 관찰해보세요.
Move Tool · 31p

이동도구 핵심 기능

이동도구는 단순히 움직이는 도구가 아니라, 선택한 레이어를 정확한 위치로 정렬하고 배치하는 데 중요한 도구입니다.

Auto Select 자동선택

  • 평소에는 꺼두고 사용하는 것이 안전합니다.
  • 원치 않는 레이어가 선택되는 실수를 줄일 수 있습니다.
  • 임시로 켜고 싶다면: Ctrl 키를 꾹 누른 상태로 사용합니다.

Show Transform Controls

  • 이동도구 상태에서 크기와 위치를 빠르게 확인할 수 있습니다.
  • 자유 변형: Ctrl + T
  • 변형 완료: Enter

정렬

  • 선택된 레이어들을 기준선에 맞춰 정렬할 수 있습니다.
  • 디자인 요소를 정확하게 배치할 때 매우 유용합니다.
  • 여러 개의 오브젝트를 깔끔하게 정리하는 데 필수입니다.

이동도구로 확인할 것

  • 지금 어떤 레이어가 선택되어 있는가
  • 현재 이동하는 대상이 맞는가
  • 레이어 위치와 정렬 상태가 의도한 결과와 같은가
Smart Object · 59p

스마트오브젝트 이해하기

스마트오브젝트는 원본을 보호하면서 작업할 수 있도록 도와주는 중요한 레이어 방식입니다. 특히 보정, 합성, 목업 작업에서 자주 사용합니다.

스마트오브젝트란?

  • 벡터 속성을 가지고 있는 레이어입니다.
  • 이미지를 작업 화면으로 바로 가져오면 자동으로 스마트오브젝트 처리되는 경우가 많습니다.
  • 크기 변형을 반복해도 화질 손상에 강합니다.

왜 좋은가?

  • 화질 보호: 줄였다가 다시 키워도 일반 레이어보다 손상이 적습니다.
  • 원본 보호: 원본 데이터를 직접 망가뜨리지 않고 작업할 수 있습니다.
  • 수정 용이: 적용한 보정을 나중에 다시 수정하거나 삭제하기 좋습니다.

주의할 점

  • 브러시, 지우개 등 원본을 훼손하는 작업은 바로 적용할 수 없습니다.
  • 즉, 원본을 보호하는 대신 일부 직접 편집은 제한됩니다.
  • 필요하면 일반 레이어로 변환한 뒤 작업해야 합니다.

수정하는 방법

  • 레이어 썸네일을 더블클릭하면 원본 또는 연결된 내용을 다시 열 수 있습니다.
  • 스마트 필터나 보정 내역이 남아 있으면 언제든 다시 조절할 수 있습니다.
  • 비파괴 편집이 필요한 작업에서 특히 강력합니다.
항목일반 레이어스마트오브젝트
크기 조절반복 변형 시 화질 저하 가능원본 정보를 유지해 비교적 안전함
직접 픽셀 편집바로 가능바로 불가능한 경우가 많음
원본 보호낮음높음
추천 작업세밀한 픽셀 수정보정, 목업
Class Summary

오늘 배운 내용 한눈에 보기

레이어

레이어는 각각 따로 편집 가능한 작업 단위이며, 그룹과 순서 이동을 통해 정리합니다.

이동도구

레이어를 선택하고 이동하고 정렬하는 도구입니다. Auto Select는 필요할 때만 잠깐 사용하는 것이 좋습니다.

스마트오브젝트

원본을 보호하면서 변형과 보정을 유연하게 처리할 수 있는 비파괴 편집용 레이어입니다.

실무 연결

보정, 합성, 목업 작업에서는 스마트오브젝트를 적극적으로 활용하면 수정과 재작업이 쉬워집니다.

Practice Tip

복습 미션

새 파일을 열고 도형 3개를 만든 뒤, 레이어 그룹 만들기, 복제하기, 순서 바꾸기, 이동도구로 정렬하기를 연습해 보세요.

마지막으로 이미지를 가져와 스마트오브젝트 상태를 확인하고, Ctrl + T로 크기를 줄였다가 다시 키워보며 차이를 확인해 보세요.
